= Element 스토리지 시스템 설정
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp Element 소프트웨어 스토리지 시스템을 설정하려면 스토리지 노드를 구성하고, 스토리지 클러스터를 만들고, 클러스터에 드라이브를 추가하는 작업이 필요합니다.  파이버 채널 네트워크를 사용하는 경우 파이버 채널 노드를 구성할 수 있습니다.
