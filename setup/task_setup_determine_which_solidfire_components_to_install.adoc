다음 표에서는 추가 구성 요소를 나열하고 이를 설치해야 하는지 여부를 나타냅니다.

* 용량 라이선스 SolidFire 스토리지 클러스터에는 Active IQ 필요합니다.

.단계
. 어떤 구성 요소를 설치해야 하는지 결정합니다.
. 설치를 완료하세요.link:../mnode/task_mnode_install.html["관리 노드를 설치합니다"] 절차.
+

NOTE: Active IQ 설정하려면 다음을 사용하세요. `--telemetry_active` Active IQ 에서 분석을 위한 데이터 수집을 활성화하기 위한 설정 스크립트의 매개변수입니다.

. NetApp Monitoring Agent 정보는 다음을 참조하세요.link:../mnode/task_mnode_enable_activeIQ.html["절차"] .
