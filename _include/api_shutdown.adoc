= 매개변수
:allow-uri-read: 


당신은 사용할 수 있습니다 `Shutdown` 클러스터의 노드를 다시 시작하거나 종료하는 방법입니다.  이 방법을 사용하면 클러스터의 단일 노드, 여러 노드 또는 모든 노드를 종료할 수 있습니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
노드
 a| 
재시작 또는 종료할 노드의 NodeID 목록입니다.
 a| 
정수 배열
 a| 
None
 a| 
예



 a| 
옵션
 a| 
클러스터에 대한 조치입니다. 가능한 값:

* restart: 클러스터를 다시 시작합니다.
* halt: 전원을 완전히 끕니다.

 a| 
끈
 a| 
재시작
 a| 
아니요

|===


== 반환 값

이 메서드에는 반환 값이 없습니다.



== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "Shutdown",
   "params": {
     "nodes": [
       2,
       3,
       4
     ],
     "option": "halt"
   },
   "id": 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
  "id" : 1,
  "result" : {
     "failed": [],
     "successful": [
          6
      [
   }
}
----
