Element 소프트웨어를 시작하는 데 도움이 되는 리소스와 도구에 대해 알아보세요.

* https://handsonlabs.netapp.com/lab/elementsw["Element를 통한 개인 클라우드 스토리지 유연성을 위한 Lab on Demand(로그인 필요)"^]: 이 랩에서는 Element 소프트웨어를 실행하는 스토리지 시스템에 적용되는 제한 없는 확장, 보장된 워크로드 성능, 스토리지 인프라 자동화의 개념을 소개합니다.
* link:task_use_demonode.html["Element Demo Node를 사용하여 저장 기능을 사용해 보세요."^]: Element Demo Node는 Element 소프트웨어의 VMware 가상 머신 버전으로, NetApp HCI 및 SolidFire 제품의 주요 스토리지 기능 중 다수를 쉽게 시연할 수 있는 방법을 제공합니다.
