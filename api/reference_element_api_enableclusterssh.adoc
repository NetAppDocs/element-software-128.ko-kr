= EnableClusterSsh
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `EnableClusterSsh` 스토리지 클러스터의 모든 노드에서 SSH 서비스를 활성화하는 방법입니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
지속
 a| 
SSH 서비스가 활성화된 상태로 유지되는 시간입니다.
 a| 
끈
 a| 
None
 a| 
예

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
결과
 a| 
스토리지 클러스터에 대한 SSH 서비스 상태, SSH가 비활성화될 때까지 남은 시간, 각 노드에 대한 SSH 서비스 상태를 포함하는 JSON 개체입니다.
 a| 
JSON 객체

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "EnableClusterSsh",
  "params": {
      "duration"  : "02:00:00.00"
  },
  "id" : 1
}
----

이 메서드는 다음 예와 유사한 응답을 반환합니다.

	"id": 1,
	"result" : {

        "enabled": false
    } ]
			}
	}
----
