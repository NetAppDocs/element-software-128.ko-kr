= 클러스터 관리자 목록
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListClusterAdmins` 클러스터의 모든 클러스터 관리자 목록을 반환하는 방법입니다.

서로 다른 수준의 권한을 가진 여러 개의 클러스터 관리자 계정이 있을 수 있습니다.  시스템에는 기본 클러스터 관리자가 한 명만 있을 수 있습니다.  기본 클러스터 관리자는 클러스터가 생성될 때 생성된 관리자입니다.  클러스터에 LDAP 시스템을 설정할 때 LDAP 관리자를 만들 수도 있습니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
숨김 표시
 a| 
SNMP 관리자와 같은 숨겨진 클러스터 관리자 사용자를 표시합니다.
 a| 
부울
 a| 
None
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
클러스터 관리자
 a| 
클러스터에 존재하는 모든 클러스터 및 LDAP 관리자에 대한 정보입니다.
 a| 
xref:reference_element_api_clusteradmin.adoc[클러스터 관리자]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "ListClusterAdmins",
   "params": {},
    "showHidden": true
   "id" : 1
}
----
