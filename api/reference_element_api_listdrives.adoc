= 드라이브 목록
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListDrives` 클러스터의 활성 노드에 있는 드라이브를 나열하는 방법입니다.  이 메서드는 볼륨 메타데이터 또는 블록 드라이브로 추가된 드라이브는 물론, 추가되지 않았지만 사용 가능한 드라이브도 반환합니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
드라이브
 a| 
클러스터의 드라이브 목록입니다.
 a| 
xref:reference_element_api_drive.adoc[운전하다]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "ListDrives",
   "params": {},
   "id" : 1
}
----
