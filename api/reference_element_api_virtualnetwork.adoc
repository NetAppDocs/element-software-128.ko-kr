= 가상 네트워크
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
virtualNetwork 객체에는 특정 가상 네트워크에 대한 정보가 포함되어 있습니다.  당신은 사용할 수 있습니다 `ListVirtualNetworks` 시스템의 모든 가상 네트워크에 대한 정보 목록을 검색하는 API 방법입니다.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
주소 블록
 a| 
현재 가상 네트워크에 할당된 주소 블록의 범위입니다.  회원:

* 사용 가능: "1"과 "0"으로 구성된 이진 문자열. "1"은 IP 주소를 사용할 수 있음을 나타내고, "0"은 IP를 사용할 수 없음을 나타냅니다.  문자열은 오른쪽에서 왼쪽으로 읽히며, 가장 오른쪽에 있는 숫자가 주소 블록 목록의 첫 번째 IP 주소입니다.
* 크기: 주소 블록의 크기입니다.
* 시작: 블록의 첫 번째 IP 주소입니다.

 a| 
JSON 객체 배열



 a| 
속성
 a| 
JSON 객체 형식의 이름-값 쌍 목록입니다.
 a| 
JSON 객체



 a| 
이름
 a| 
가상 네트워크에 할당된 이름입니다.
 a| 
끈



 a| 
넷마스크
 a| 
가상 네트워크에 대한 넷마스크의 IP 주소입니다.
 a| 
끈



 a| 
svip
 a| 
가상 네트워크의 저장소 IP 주소입니다.
 a| 
끈



 a| 
게이트웨이
 a| 
가상 네트워크에 사용되는 게이트웨이입니다.
 a| 
끈



 a| 
가상네트워크ID
 a| 
가상 네트워크의 고유 식별자입니다.
 a| 
정수



 a| 
가상네트워크태그
 a| 
VLAN 태그 식별자입니다.
 a| 
정수

|===
