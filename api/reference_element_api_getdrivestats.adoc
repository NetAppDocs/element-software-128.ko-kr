= 드라이브 통계 가져오기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetDriveStats` 단일 주행에 대한 높은 수준의 활동 측정값을 얻는 방법입니다.  값은 클러스터에 드라이브를 추가한 이후부터 누적됩니다.  일부 값은 블록 드라이브에만 적용됩니다.  이 방법을 실행하면 블록 또는 메타데이터 드라이브 유형에 대한 통계 데이터가 반환됩니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
드라이브ID
 a| 
요청에 대한 드라이브의 ID입니다.
 a| 
정수
 a| 
None
 a| 
예

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
드라이브 통계
 a| 
지정된 driveID에 대한 드라이브 활동 정보입니다.
 a| 
xref:reference_element_api_drivestats.adoc[드라이브 통계]

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "GetDriveStats",
   "params": {
     "driveID": 3
   },
   "id" : 1
}
----
