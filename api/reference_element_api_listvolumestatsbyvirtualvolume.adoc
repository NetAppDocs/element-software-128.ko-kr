= 목록볼륨통계별가상볼륨
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListVolumeStatsByVirtualVolume` 가상 볼륨과 연관된 시스템의 모든 볼륨에 대한 볼륨 통계를 나열하는 방법입니다.  통계는 볼륨 생성 시점부터 누적됩니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
가상 볼륨 ID
 a| 
정보를 검색할 하나 이상의 가상 볼륨 ID 목록입니다.  이 매개변수를 지정하면 해당 메서드는 해당 가상 볼륨에 대한 정보만 반환합니다.
 a| 
UUID 문자열 배열
 a| 
아니요
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
볼륨통계
 a| 
시스템의 각 가상 볼륨에 대한 활동 정보가 포함된 개체 목록입니다.
 a| 
xref:reference_element_api_volumestats.adoc[볼륨통계]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "ListVolumeStatsByVirtualVolume",
  "params": {},
  "id": 1
}
----
