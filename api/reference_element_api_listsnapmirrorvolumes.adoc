= ListSnapMirrorVolumes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element 소프트웨어 웹 UI는 다음을 사용합니다. `ListSnapMirrorVolumes` 원격 ONTAP 시스템에서 사용 가능한 모든 SnapMirror 볼륨을 나열하는 방법입니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
스냅미러엔드포인트ID
 a| 
지정된 엔드포인트 ID와 연결된 볼륨만 나열합니다.  엔드포인트 ID가 제공되지 않으면 시스템은 알려진 모든 SnapMirror 엔드포인트의 볼륨을 나열합니다.
 a| 
정수
 a| 
None
 a| 
아니요



 a| 
v서버
 a| 
지정된 Vserver에 호스팅된 볼륨을 나열합니다.  Vserver는 "데이터" 유형이어야 합니다.
 a| 
끈
 a| 
None
 a| 
아니요



 a| 
이름
 a| 
지정된 이름을 가진 ONTAP 볼륨만 나열합니다.
 a| 
끈
 a| 
None
 a| 
아니요



 a| 
유형
 a| 
지정된 유형의 ONTAP 볼륨만 나열합니다. 가능한 값:

* rw: 읽기-쓰기 볼륨
* ls: 부하 공유 볼륨
* dp: 데이터 보호 볼륨

 a| 
끈
 a| 
None
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
스냅미러볼륨
 a| 
ONTAP 스토리지 시스템에서 사용 가능한 SnapMirror 볼륨 목록입니다.
 a| 
xref:reference_element_api_snapmirrorvolume.adoc[스냅미러볼륨]정렬

|===
