= QoSPolicy
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
QoSPolicy 개체에는 Element 소프트웨어를 실행하는 스토리지 클러스터의 QoS 정책에 대한 정보가 포함되어 있습니다.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
qos정책ID
 a| 
스토리지 클러스터에서 자동으로 할당하는 QoSPolicy에 대한 고유한 정수 식별자입니다.
 a| 
정수



 a| 
이름
 a| 
QoS 정책의 이름입니다.  예를 들어 금, 백금, 은 등입니다.
 a| 
끈



 a| 
qos
 a| 
이 정책이 나타내는 QoS 설정입니다.
 a| 
xref:reference_element_api_qos.adoc[서비스 품질]



 a| 
볼륨 ID
 a| 
이 정책과 관련된 볼륨 목록입니다.
 a| 
정수 배열

|===
