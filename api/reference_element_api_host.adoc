= 호스트(가상 볼륨)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
호스트 개체에는 가상 볼륨 호스트에 대한 정보가 포함되어 있습니다.  당신은 사용할 수 있습니다 `ListVirtualVolumeHosts` 모든 가상 볼륨 호스트에 대한 정보를 얻는 방법입니다.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
바인딩
 a| 
가상 볼륨 호스트의 바인딩을 설명하는 개체 목록입니다.
 a| 
정수 배열



 a| 
클러스터ID
 a| 
이 호스트가 연결된 클러스터의 고유 ID입니다.
 a| 
UUID



 a| 
호스트 주소
 a| 
가상 볼륨 호스트의 IP 주소 또는 DNS 이름입니다.
 a| 
끈



 a| 
개시자 이름
 a| 
가상 볼륨 호스트에 대한 개시자 IQN 목록입니다.
 a| 
문자열 배열



 a| 
가상볼륨호스트ID
 a| 
이 가상 볼륨 호스트의 고유 ID입니다.
 a| 
UUID



 a| 
visibleProtocolEndpointIDs
 a| 
이 호스트에서 볼 수 있는 프로토콜 엔드포인트의 ID 목록입니다.
 a| 
UUID 배열

|===
