= SSL 인증서 제거
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `RemoveSSLCertificate` 클러스터의 스토리지 노드에 대한 사용자 SSL 인증서와 개인 키를 제거하는 방법입니다.  인증서와 개인 키가 제거되면 저장 노드는 기본 인증서와 개인 키를 사용하도록 구성됩니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드에는 반환 값이 없습니다.



== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
    "method" : "RemoveSSLCertificate",
    "params" : {},
    "id" : 3
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
    "id" : 3,
    "result" : {}
}
----
