= DeleteSnapMirrorEndpoints
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element 웹 UI는 다음을 사용합니다. `DeleteSnapMirrorEndpoints` 시스템에서 하나 이상의 SnapMirror 엔드포인트를 삭제합니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
snapMirrorEndpointIDs
 a| 
삭제할 SnapMirror 엔드포인트의 ID 배열입니다.
 a| 
정수 배열
 a| 
None
 a| 
예

|===


== 반환 값

이 메서드에는 반환 값이 없습니다.

10.0
