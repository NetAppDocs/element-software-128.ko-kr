= 볼륨 액세스 그룹 삭제
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
사용할 수 있습니다 `DeleteVolumeAccessGroup` 볼륨 액세스 그룹을 삭제합니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
볼륨액세스그룹ID
 a| 
삭제할 볼륨 액세스 그룹의 ID입니다.
 a| 
정수
 a| 
None
 a| 
예



 a| 
deleteOrphanInitiators
 a| 
개시자 객체를 삭제할지 여부를 지정합니다. 가능한 값:

* true: 볼륨 액세스 그룹에서 제거된 후 개시자 개체를 삭제합니다.
* false: 볼륨 액세스 그룹에서 제거된 후에는 개시자 객체를 삭제하지 마십시오. 이것이 기본값입니다.

 a| 
부울
 a| 
거짓
 a| 
아니요



 a| 
힘
 a| 
이 플래그를 추가하면 가상 네트워크 ID 또는 태그가 있더라도 볼륨 액세스 그룹이 삭제됩니다. 가능한 값:

* true: 볼륨 액세스 그룹이 삭제됩니다.
* false: 기본값.  가상 네트워크 ID 또는 태그가 있는 경우 볼륨 액세스 그룹을 삭제하지 마세요.

 a| 
부울
 a| 
거짓
 a| 
아니요

|===


== 반환 값

이 메서드에는 반환 값이 없습니다.



== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "DeleteVolumeAccessGroup",
   "params": {
          "force": true,
	  "volumeAccessGroupID"  : 3
   },
   "id" : 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
   "id" : 1,
   "result": {}
}
----
