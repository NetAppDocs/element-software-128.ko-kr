= QuiesceSnapMirror관계
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element 소프트웨어 웹 UI는 다음을 사용합니다. `QuiesceSnapMirrorRelationship` SnapMirror 관계에 대한 향후 데이터 전송을 비활성화하는 방법입니다.  전송이 진행 중이면 전송이 완료될 때까지 관계 상태가 "중단"으로 전환됩니다.  현재 전송이 중단되면 다시 시작되지 않습니다.  관계에 대한 데이터 전송을 다시 활성화하려면 다음을 사용하십시오. `ResumeSnapMirrorRelationship` API 방식.
