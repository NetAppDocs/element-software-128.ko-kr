= 스냅미러정책
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
snapMirrorPolicy 개체에는 ONTAP 시스템에 저장된 SnapMirror 정책에 대한 정보가 포함되어 있습니다.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
스냅미러엔드포인트ID
 a| 
대상 ONTAP 시스템의 ID입니다.
 a| 
정수



 a| 
정책 이름
 a| 
정책에 할당된 고유한 이름입니다.
 a| 
끈



 a| 
정책 유형
 a| 
정책의 유형. 가능한 값:

* 비동기 미러
* 미러_볼트

 a| 
끈



 a| 
논평
 a| 
SnapMirror 정책과 관련된 사람이 읽을 수 있는 설명입니다.
 a| 
끈



 a| 
전송 우선 순위
 a| 
SnapMirror 전송이 실행되는 우선 순위입니다. 가능한 값:

* normal: 기본 우선순위.  이러한 전송은 대부분의 낮은 우선순위 전송보다 먼저 예약됩니다.
* 낮음: 이러한 전송은 우선순위가 가장 낮으며 대부분의 일반 우선순위 전송 이후에 예약됩니다.

 a| 
끈



 a| 
정책 규칙
 a| 
정책 규칙을 설명하는 객체 목록입니다.
 a| 
xref:reference_element_api_snapmirrorpolicyrule.adoc[스냅미러정책규칙]정렬



 a| 
총 유지 횟수
 a| 
정책의 모든 규칙에 대한 총 보존 횟수입니다.
 a| 
정수



 a| 
총 규칙
 a| 
정책의 총 규칙 수입니다.
 a| 
정수



 a| 
vserverName
 a| 
SnapMirror 정책에 대한 Vserver의 이름입니다.
 a| 
끈

|===
