= 리스트스냅샷
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
사용할 수 있습니다 `ListSnapshots` 볼륨에서 촬영된 각 스냅샷의 속성을 반환합니다.

이 메서드가 소스 클러스터에서 호출되면 대상 클러스터에 있는 스냅샷에 대한 정보가 소스 클러스터에 표시됩니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
볼륨ID
 a| 
볼륨의 스냅샷을 검색합니다.  volumeID가 제공되지 않으면 모든 볼륨의 모든 스냅샷이 반환됩니다.
 a| 
정수
 a| 
None
 a| 
아니요



 a| 
스냅샷ID
 a| 
개별 스냅샷 ID에 대한 정보를 검색합니다.
 a| 
정수
 a| 
None
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
스냅샷
 a| 
각 볼륨의 각 스냅샷에 대한 정보입니다.  volumeID가 제공되지 않으면 모든 볼륨의 모든 스냅샷이 반환됩니다.  그룹 내의 스냅샷은 그룹 ID와 함께 반환됩니다.
 a| 
xref:reference_element_api_snapshot.adoc[스냅 사진]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "ListSnapshots",
   "params": {
             "volumeID": "1"
       },
   "id" : 1
}
----
