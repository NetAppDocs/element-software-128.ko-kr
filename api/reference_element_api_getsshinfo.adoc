= GetSshInfo
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetSshInfo` 단일 노드에서 SSH 서비스의 상태를 쿼리하는 방법입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
결과
 a| 
이 노드의 SSH 서비스 상태입니다.
 a| 
부울

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
    "method" : "GetSshInfo",
    "params" : {},
    "id" : 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
    "id": 1,
    "result": {
        "enabled": false
    }
}
----
