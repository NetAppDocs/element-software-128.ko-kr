= 세션(파이버 채널)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
세션 개체에는 클러스터에서 볼 수 있는 각 파이버 채널 세션에 대한 정보와 해당 세션이 볼 수 있는 대상 포트에 대한 정보가 포함되어 있습니다.  이 정보는 다음을 사용하여 검색할 수 있습니다. `ListFibreChannelSessions` API 방식.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
개시자WWPN
 a| 
대상 포트에 로그인한 초기자의 WWPN(World Wide Port Name)입니다.
 a| 
끈



 a| 
노드ID
 a| 
파이버 채널 세션을 소유한 노드입니다.
 a| 
정수



 a| 
창시자
 a| 
이 파이버 채널 세션의 서버 개시자에 대한 정보입니다.  회원:

* 별칭: 개시자에게 지정된 친근한 이름입니다.
* 속성: 이 개시자의 속성입니다.
* initiatorID: 이 개시자의 ID입니다.
* initiatorName: 이 개시자의 이름입니다.
* volumeAccessGroups: 이 개시자와 연관된 볼륨 액세스 그룹 목록입니다.

 a| 
JSON 객체



 a| 
서비스ID
 a| 
이 세션에 관련된 대상 포트의 서비스 ID입니다.
 a| 
정수



 a| 
타겟WWPN
 a| 
이 세션에 관련된 대상 포트의 WWPN입니다.
 a| 
끈



 a| 
볼륨액세스그룹ID
 a| 
initiatorWWPN이 속한 볼륨 액세스 그룹의 ID입니다.  볼륨 액세스 그룹에 없으면 이 값은 null입니다.
 a| 
정수

|===
