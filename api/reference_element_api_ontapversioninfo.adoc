= ontapVersionInfo
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
ontapVersionInfo 개체에는 SnapMirror 관계에서 ONTAP 클러스터의 API 버전에 대한 정보가 포함되어 있습니다.  Element 웹 UI는 다음을 사용합니다. `GetOntapVersionInfo` 이 정보를 얻기 위한 API 메소드입니다.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
스냅미러엔드포인트ID
 a| 
대상 ONTAP 시스템의 ID입니다.
 a| 
정수



 a| 
클라이언트 API 주요 버전
 a| 
Element API 클라이언트에서 사용되는 ONTAP API 주요 버전입니다.
 a| 
끈



 a| 
클라이언트 API 부버전
 a| 
Element API 클라이언트에서 사용 중인 ONTAP API 하위 버전입니다.
 a| 
끈



 a| 
ontapAPIMajorVersion
 a| 
ONTAP 시스템이 지원하는 현재 API 주요 버전입니다.
 a| 
끈



 a| 
ontapAPIMinorVesion
 a| 
ONTAP 시스템이 지원하는 현재 API 마이너 버전입니다.
 a| 
끈



 a| 
온탭버전
 a| 
ONTAP 클러스터에서 실행 중인 현재 소프트웨어 버전입니다.
 a| 
끈

|===
