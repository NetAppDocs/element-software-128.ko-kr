= 계정별 볼륨 통계 목록
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListVolumeStatsByAccount` 모든 계정에 대한 높은 수준의 볼륨 활동 측정값을 나열하는 방법입니다.  값은 계정이 소유한 모든 볼륨의 합계입니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
가상 볼륨 포함
 a| 
가상 볼륨은 기본적으로 응답에 포함됩니다.  가상 볼륨을 제외하려면 false로 설정합니다.
 a| 
부울
 a| 
true
 a| 
아니요



 a| 
계정
 a| 
볼륨 통계를 반환할 계정 ID 목록입니다.  생략하면 모든 계정에 대한 통계가 반환됩니다.
 a| 
정수 배열
 a| 
None
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
볼륨통계
 a| 
각 계정의 볼륨 활동 정보 목록입니다.*참고:* 각 항목의 볼륨 ID 멤버는 0입니다. 값은 해당 계정이 소유한 모든 볼륨의 합계를 나타냅니다.
 a| 
xref:reference_element_api_volumestats.adoc[볼륨통계]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
"method": "ListVolumeStatsByAccount",
  "params": {"accounts": [3]},
  "id": 1
}
----
