= 테스트 드라이브
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `TestDrives` 노드의 모든 드라이브에 대해 하드웨어 검증을 실행하는 방법입니다.  이 방법은 드라이브의 하드웨어 오류를 감지하고 검증 테스트 결과를 보고합니다.



== 매개변수

당신은 단지 사용할 수 있습니다 `TestDrives` 클러스터에서 "활성" 상태가 아닌 노드에 대한 방법입니다.


NOTE: 이 테스트는 약 10분 정도 걸립니다.

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
힘
 a| 
노드의 드라이브를 테스트하려면 true로 설정합니다.
 a| 
부울
 a| 
None
 a| 
예



 a| 
분
 a| 
테스트를 실행하는 데 걸리는 시간을 분 단위로 지정합니다.
 a| 
정수
 a| 
10
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
세부
 a| 
테스트 작업의 성공 또는 실패에 대한 정보입니다.
 a| 
JSON 객체

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "TestDrives",
   "params": {
              "force": true,
              "minutes" : 10
             },
   "id" : 1
}
----


== 응답 예시

이 메서드는 노드의 각 드라이브에 대한 테스트 결과가 포함된 테이블을 반환합니다.
