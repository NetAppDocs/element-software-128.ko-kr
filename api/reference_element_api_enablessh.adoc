= EnableSsh
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `EnableSsh` 단일 노드에 대해 Secure Shell(SSH) 서비스를 활성화하는 방법입니다.  이 방법은 클러스터 전체의 SSH 시간 초과 기간에 영향을 미치지 않으며, 글로벌 SSH 시간 초과로 인해 노드의 SSH가 비활성화되는 것을 면제하지 않습니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
활성화됨
 a| 
이 노드의 SSH 서비스 상태입니다.
 a| 
부울

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "EnableSsh",
  "params": {
     },
  "id" : 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
  "id" : 1,
  "result" : {"enabled": true}
}
----
