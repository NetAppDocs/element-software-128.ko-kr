= 테스트커넥트앙상블
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `TestConnectEnsemble` 지정된 데이터베이스 앙상블과의 연결성을 확인하는 방법입니다.  기본적으로 노드가 연결된 클러스터의 앙상블을 사용합니다.  혹은 연결성을 테스트하기 위해 다른 앙상블을 제공할 수도 있습니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


| 앙상블 | 연결성 테스트를 위한 앙상블 노드 클러스터 IP 주소의 쉼표로 구분된 목록입니다. | 끈 | None | 아니요 
|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


| 세부  a| 
반환된 객체:

* `nodes`: (객체) 테스트의 각 앙상블 노드 목록과 테스트 결과입니다.
* `duration`: (문자열) 테스트를 실행하는 데 필요한 시간입니다.
* `result`: (문자열) 전체 테스트의 결과입니다.

| JSON 객체 
|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "TestConnectEnsemble",
  "params": {},
  "id": 1
}
----
