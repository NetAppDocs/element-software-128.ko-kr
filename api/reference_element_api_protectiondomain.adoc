= 보호 도메인
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
protectionDomain 개체에는 보호 도메인의 이름과 유형 세부 정보가 포함되어 있습니다.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
보호 도메인 이름
 a| 
보호 도메인의 이름입니다.
 a| 
끈



 a| 
보호 도메인 유형
 a| 
보호 도메인의 유형입니다. 가능한 값:

* 섀시: 단일 섀시에 있는 모든 저장 노드.
* 사용자 정의: 단일 고객 정의 보호 도메인의 모든 스토리지 노드입니다.

 a| 
끈

|===
