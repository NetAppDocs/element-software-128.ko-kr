= GetLldpInfo
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetLldpInfo` 스토리지 클러스터의 각 노드 또는 개별 스토리지 노드에 대한 LLDP(Link Layer Discovery Protocol) 구성을 가져오는 방법입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
lldp정보
 a| 
스토리지 클러스터의 각 노드에 대한 섀시, 인터페이스 및 이웃 LLDP 설정에 대한 정보입니다.
 a| 
JSON 객체

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "GetLldpInfo",
   "id" : 1
}
----


== 응답 예시

이 응답 예시는 길이가 길기 때문에 보충 주제에 문서화되어 있습니다.
