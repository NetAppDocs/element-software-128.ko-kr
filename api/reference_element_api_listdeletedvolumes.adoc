= 삭제된 볼륨 목록
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListDeletedVolumes` 삭제 표시가 되어 시스템에서 제거된 볼륨 목록을 검색하는 방법입니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
가상 볼륨 포함
 a| 
기본적으로 가상 볼륨이 응답에 포함됩니다.  가상 볼륨을 제외하려면 false로 설정합니다.
 a| 
부울
 a| 
true
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
볼륨
 a| 
삭제된 볼륨 목록.
 a| 
xref:reference_element_api_volume.adoc[용량]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "ListDeletedVolumes",
   "params": {},
   "id" : 1
}
----
