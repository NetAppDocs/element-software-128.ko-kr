= 클러스터페어
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
clusterPair 객체에는 로컬 클러스터와 쌍을 이루는 클러스터에 대한 정보가 포함되어 있습니다.  ListClusterPairs 메서드를 사용하면 로컬 클러스터의 clusterPair 객체 목록을 검색할 수 있습니다.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
클러스터 이름
 a| 
쌍의 다른 클러스터의 이름입니다.
 a| 
끈



 a| 
클러스터페어ID
 a| 
쌍의 각 클러스터에 부여된 고유한 ID입니다.
 a| 
정수



 a| 
클러스터페어UUID
 a| 
클러스터 쌍에 대한 범용 고유 식별자입니다.
 a| 
끈



 a| 
UUID
 a| 
클러스터 쌍의 원격 클러스터에 대한 고유 식별자입니다.
 a| 
정수



 a| 
숨어 있음
 a| 
클러스터 간 지연 시간(밀리초)입니다.
 a| 
정수



 a| 
엠비피
 a| 
페어링된 클러스터의 관리 연결에 대한 IP 주소입니다.
 a| 
끈



 a| 
상태
 a| 
쌍을 이루는 클러스터 간의 연결 상태입니다. 가능한 값:

* 구성되지 않음
* 연결됨
* 잘못 구성됨
* 연결이 끊김

 a| 
끈



 a| 
버전
 a| 
쌍의 다른 클러스터의 요소 버전입니다.
 a| 
끈

|===
