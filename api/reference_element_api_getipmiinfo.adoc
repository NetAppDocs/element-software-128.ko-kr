= GetIpmiInfo
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetIpmiInfo` 시스템에서 모니터링하는 노드 팬, 흡기 및 배기 온도, 전원 공급 장치에 대한 센서(객체)에 대한 자세한 보고서를 표시하는 방법입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
센서
 a| 
노드 내의 각 센서로부터 얻은 자세한 정보입니다.
 a| 
JSON 객체 배열

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "GetIpmiInfo",
  "params": {},
  "id" : 1
}
----
