= ListIdpConfigurations
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListIdpConfigurations` 타사 IdP에 대한 구성을 나열하는 방법입니다.  선택적으로 다음 중 하나를 제공할 수 있습니다. `enabledOnly` 현재 활성화된 IdP 구성이나 IdP 메타데이터 UUID 또는 IdP 이름을 검색하여 특정 IdP 구성에 대한 정보를 쿼리하는 플래그입니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
활성화됨만
 a| 
현재 활성화된 IdP 구성을 반환하기 위해 결과를 필터링합니다.
 a| 
부울
 a| 
None
 a| 
아니요



 a| 
idp구성ID
 a| 
타사 IdP 구성을 위한 UUID입니다.
 a| 
UUID
 a| 
None
 a| 
아니요



 a| 
IDP 이름
 a| 
특정 IdP 이름에 대한 IdP 구성 정보를 검색합니다.
 a| 
끈
 a| 
None
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
idpConfigInfos
 a| 
타사 IdP 구성에 대한 정보입니다.
 a| 
link:reference_element_api_idpconfiginfo.html["idpConfigInfo"]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "ListIdpConfigurations",
  "params": {}
}
----
