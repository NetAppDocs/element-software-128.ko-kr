= GetBinAssignmentProperties
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetBinAssignmentProperties` 데이터베이스에서 빈 할당 속성을 검색하는 방법입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
속성
 a| 
데이터베이스의 모든 현재 빈 할당에 대한 속성을 자세히 설명합니다.
 a| 
binAssignmentProperties 배열

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
    "method": "GetBinAssignmentProperties",
    "params": {
    },
    "id": 1
}
----
