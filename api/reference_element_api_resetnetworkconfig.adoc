= 네트워크 구성 재설정
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ResetNetworkConfig` 개별 노드의 네트워크 구성 문제를 해결하는 데 도움이 되는 방법입니다.  이 방법은 개별 노드의 네트워크 구성을 공장 기본 설정으로 재설정합니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드에는 반환 값이 없습니다.



== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "ResetNetworkConfig",
   "params": {},
   "id" : 1
}
----


== 응답 예시

이 메서드는 응답을 반환하지 않습니다.
