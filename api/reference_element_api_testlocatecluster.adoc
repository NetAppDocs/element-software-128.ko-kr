= 테스트로케이트클러스터
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `TestLocateCluster` 클러스터 구성에 지정된 클러스터를 노드가 찾을 수 있는지 확인하는 방법입니다.  출력은 클러스터가 생성되었는지 검증하고 클러스터 앙상블의 노드를 나열합니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
세부
 a| 
테스트 작업의 성공 또는 실패에 대한 정보입니다.
 a| 
JSON 객체

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "TestLocateCluster",
   "params": {},
   "id" : 1
}
----
