= ListAuthSessionsByClusterAdmin
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListAuthSessionsByClusterAdmin` 지정된 것과 관련된 모든 인증 세션을 나열하는 방법 `ClusterAdminID` .  지정된 경우 `ClusterAdminID` 사용자 그룹에 매핑하면 해당 그룹의 모든 멤버에 대한 모든 인증 세션이 나열됩니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
클러스터 관리자 ID
 a| 
클러스터 관리자를 위한 고유 식별자입니다.
 a| 
정수
 a| 
None
 a| 
예

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
세션
 a| 
인증 세션에 대한 세션 정보 목록입니다.
 a| 
link:reference_element_api_authsessioninfo.html["인증 세션 정보"]

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
    "method": "ListAuthSessionsByClusterAdmin",
    "clusterAdminID": 1
}
----

      }
    ]
}
