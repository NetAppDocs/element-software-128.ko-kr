= 가상볼륨 바인딩 목록
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListVirtualVolumeBindings` 프로토콜 엔드포인트에 바인딩된 클러스터의 모든 가상 볼륨 목록을 가져오는 방법입니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
virtualVolumeBindingIDs
 a| 
정보를 검색할 가상 볼륨 바인딩 ID 목록입니다.  이 매개변수를 생략하면 메서드는 모든 가상 볼륨 바인딩에 대한 정보를 반환합니다.
 a| 
정수 배열
 a| 
None
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
바인딩
 a| 
프로토콜 엔드포인트에 바인딩된 클러스터의 모든 가상 볼륨을 설명하는 개체 목록입니다.
 a| 
xref:reference_element_api_binding_vvols.adoc[제본]

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "ListVirtualVolumeBindings",
    "params": {
    },
    "id": 1
}
----
