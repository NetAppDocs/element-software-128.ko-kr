= 보호 계획 허용 범위
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
protectionSchemeTolerance 개체에는 특정 보호 체계에 대한 스토리지 클러스터가 장애 발생 후에도 계속해서 데이터를 읽고 쓸 수 있는지 여부에 대한 정보가 포함되어 있습니다.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
보호 계획
 a| 
이 스토리지 클러스터의 현재 보호 체계입니다.  가능한 값은 doubleHelix뿐입니다.
 a| 
끈



 a| 
지속 가능한 블록 데이터 실패
 a| 
연관된 보호 체계에 대한 블록 데이터 가용성을 잃지 않고 발생할 수 있는 동시 실패의 현재 수입니다.
 a| 
정수



 a| 
지속 가능한 메타데이터 실패
 a| 
연관된 보호 체계에 대한 메타데이터 가용성을 잃지 않고 발생할 수 있는 동시 실패의 현재 수입니다.
 a| 
정수

|===
