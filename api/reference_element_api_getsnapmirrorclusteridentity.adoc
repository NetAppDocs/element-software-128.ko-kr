= GetSnapMirrorClusterIdentity
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element 소프트웨어 웹 UI는 다음을 사용합니다. `GetSnapMirrorClusterIdentity` ONTAP 클러스터에 대한 ID 정보를 가져옵니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
스냅미러엔드포인트ID
 a| 
제공된 경우 시스템은 지정된 snapMirrorEndpointID를 사용하여 엔드포인트의 클러스터 ID를 나열합니다.  제공되지 않으면 시스템은 알려진 모든 SnapMirror 엔드포인트의 클러스터 ID를 나열합니다.
 a| 
정수
 a| 
None
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
스냅미러클러스터아이덴티티
 a| 
SnapMirror 엔드포인트의 클러스터 ID 목록입니다.
 a| 
xref:reference_element_api_snapmirrorclusteridentity.adoc[스냅미러클러스터아이덴티티]정렬

|===
