= 인증서 세부 정보
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
그만큼 `certificateDetails` 객체에는 보안 인증서에 대한 디코딩된 정보가 포함되어 있습니다.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
`issuer`
 a| 
발급자의 이름.
 a| 
끈



 a| 
`modulus`
 a| 
공개 키의 모듈러스.
 a| 
끈



 a| 
`notAfter`
 a| 
인증서의 만료일.
 a| 
ISO 8601 문자열



 a| 
`notBefore`
 a| 
인증서 시작 날짜.
 a| 
ISO 8601 문자열



 a| 
`serial`
 a| 
인증서 일련번호.
 a| 
끈



 a| 
`sha1Fingerprint`
 a| 
인증서의 DER로 인코딩된 버전의 다이제스트입니다.
 a| 
끈



 a| 
`subject`
 a| 
주제 이름.
 a| 
끈

|===
