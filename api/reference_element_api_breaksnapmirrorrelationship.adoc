= BreakSnapMirror관계
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element 웹 UI는 다음을 사용합니다. `BreakSnapMirrorRelationship` SnapMirror 관계를 끊는 방법.  SnapMirror 관계가 끊어지면 대상 볼륨이 읽기-쓰기가 가능하고 독립적이 되며 소스에서 분리될 수 있습니다.  당신은 관계를 다시 설정할 수 있습니다 `ResyncSnapMirrorRelationship` API 방식.  이 방법을 사용하려면 ONTAP 클러스터를 사용할 수 있어야 합니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
스냅미러엔드포인트ID
 a| 
Element 스토리지 클러스터와 통신하는 원격 ONTAP 스토리지 시스템의 엔드포인트 ID입니다.
 a| 
정수
 a| 
None
 a| 
예



 a| 
목적지볼륨
 a| 
SnapMirror 관계의 대상 볼륨입니다.
 a| 
xref:reference_element_api_snapmirrorvolumeinfo.adoc[스냅미러볼륨정보]
 a| 
None
 a| 
예

|===
