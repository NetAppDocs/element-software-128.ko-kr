= 클러스터 쌍 목록
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListClusterPairs` 현재 클러스터와 쌍을 이루는 모든 클러스터를 나열하는 방법입니다.  이 메서드는 현재 페어링에 대한 통계, 클러스터 페어링의 연결 및 대기 시간(밀리초) 등 활성 및 보류 중인 클러스터 페어링에 대한 정보를 반환합니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
클러스터 쌍
 a| 
각 쌍의 클러스터에 대한 정보입니다.
 a| 
xref:reference_element_api_clusterpair.adoc[클러스터페어]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "ListClusterPairs",
   "params": {
      },
   "id" : 1
}
----
