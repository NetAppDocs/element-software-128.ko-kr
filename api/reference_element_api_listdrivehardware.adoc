= ListDriveHardware
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListDriveHardware` 노드에 연결된 모든 드라이브를 나열하는 방법입니다.  개별 노드에서 이 방법을 사용하면 드라이브 하드웨어 정보가 반환됩니다.  클러스터 마스터 노드 MVIP에서 이 방법을 사용하면 모든 노드의 모든 드라이브에 대한 정보가 반환됩니다.



== 매개변수


NOTE: 메서드 응답의 "securitySupported": true 줄은 드라이브가 암호화가 가능하다는 것을 의미하지 않습니다. 단지 보안 상태를 쿼리할 수 있다는 것을 의미합니다.  모델 번호가 "-NE"로 끝나는 노드 유형이 있는 경우, 이러한 드라이브에서 보안 기능을 활성화하는 명령은 실패합니다.

이 메서드에는 다음과 같은 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
힘
 a| 
모든 노드에서 이 메서드를 실행하려면 true로 설정합니다.
 a| 
부울
 a| 
None
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
드라이브하드웨어
 a| 
노드에 대한 드라이브 하드웨어 정보가 반환되었습니다.
 a| 
JSON 객체 배열

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
"method": "ListDriveHardware",
"params": {},
"id" : 1
}
----
