= ListStorageContainers
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListStorageContainers` 시스템에 알려진 모든 가상 볼륨 저장 컨테이너에 대한 정보를 검색하는 방법입니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
storageContainerIDs
 a| 
정보를 검색할 저장 컨테이너 ID 목록입니다.  이 매개변수를 생략하면 메서드는 시스템의 모든 저장 컨테이너에 대한 정보를 반환합니다.
 a| 
UUID 배열
 a| 
None
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
저장용기
 a| 
시스템의 모든 저장 컨테이너에 대한 정보가 포함된 개체 목록입니다.
 a| 
xref:reference_element_api_storagecontainer.adoc[저장용기]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "ListStorageContainers",
   "params": {
      "storageContainerIDs": ["efda8307-b916-4424-979e-658a3f16894d"]
   },
   "id" : 1
}
----

이 메서드는 다음 예와 유사한 응답을 반환합니다.

    "storageContainers": [

        "targetSecret": "g38}zWBK%206jQr~",

      }
    ]
  }
}
----
