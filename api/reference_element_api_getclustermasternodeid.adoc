= 클러스터마스터노드ID 가져오기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetClusterMasterNodeID` 클러스터 전체 관리 작업을 수행하고 스토리지 가상 IP 주소(SVIP)와 관리 가상 IP 주소(MVIP)를 보유하는 노드의 ID를 검색하는 방법입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
노드ID
 a| 
마스터 노드의 ID입니다.
 a| 
정수

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "GetClusterMasterNodeID",
  "params": {},
  "id" : 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
  "id" : 1
  "result": {
    "nodeID": 1
   }
}
----
