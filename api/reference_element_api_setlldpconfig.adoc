= SetLldpConfig
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `SetLldpConfig` 스토리지 클러스터에 대한 LLDP(Link Layer Discovery Protocol) 설정을 구성하는 방법입니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
다른 프로토콜 활성화
 a| 
CDP, FDP, EDP, SONMP 등 다른 검색 프로토콜의 자동 사용을 활성화합니다.
 a| 
부울
 a| 
true
 a| 
아니요



 a| 
인에이블메드
 a| 
미디어 엔드포인트 검색(LLDP-MED)을 활성화합니다.
 a| 
부울
 a| 
거짓
 a| 
아니요



 a| 
enableLldp
 a| 
LLDP를 활성화하거나 비활성화합니다.
 a| 
부울
 a| 
true
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
lldp구성
 a| 
새로 변경된 설정을 포함하여 현재 스토리지 클러스터 LLDP 구성에 대한 정보입니다.
 a| 
JSON 객체

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "id": 3920,
  "method":"SetLldpConfig",
    "params": {
        "lldpConfig": {
            "enableMed": true
        }
    }
  }
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
  "id": 3920,
  "result": {
    "lldpConfig": {
        "enableLldp": true,
        "enableMed": true,
        "enableOtherProtocols": true
     }
  }
}
----
