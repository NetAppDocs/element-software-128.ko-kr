= 원격 연결 테스트
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
사용할 수 있습니다 `TestRemoteConnectivity` 원격 클러스터의 각 노드에 ping을 보내고 원격 앙상블 데이터베이스 연결을 확인하는 방법입니다.  이 방법으로 유용한 결과를 얻으려면 클러스터를 쌍으로 연결해야 합니다.  원격 데이터베이스 연결이 실패하면 시스템의 응답에 예외가 나열됩니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
세부
 a| 
각 노드에 대한 개별 ping 응답 시간.
 a| 
JSON 객체

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "TestRemoteConnectivity",
    "params": {
      "force": "true"
    },
    "id": 1
}
----
