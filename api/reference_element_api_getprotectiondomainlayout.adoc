= 보호 도메인 레이아웃 가져오기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetProtectionDomainLayout` 각 노드가 속한 섀시와 사용자 정의 보호 도메인을 포함하여 클러스터에 대한 모든 보호 도메인 정보를 반환하는 방법입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
보호 도메인 레이아웃
 a| 
각 노드와 연관된 보호 도메인의 목록입니다.
 a| 
JSON 목록link:reference_element_api_nodeprotectiondomains.html["노드 보호 도메인"] 사물.

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "GetProtectionDomainLayout",
  "params": {},
  "id" : 1
}
----
