= 로그인 배너 받기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetLoginBanner` 사용자가 Element 웹 인터페이스에 로그인할 때 표시되는 현재 활성화된 이용 약관 배너를 가져오는 방법입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
기치
 a| 
이용 약관 배너의 현재 텍스트입니다.  배너가 비활성화된 경우에도 이 값에는 텍스트가 포함될 수 있습니다.
 a| 
끈



 a| 
활성화됨
 a| 
이용 약관 배너의 상태입니다. 가능한 값:

* true: 웹 인터페이스 로그인 시 이용 약관 배너가 표시됩니다.
* false: 웹 인터페이스 로그인 시 이용 약관 배너가 표시되지 않습니다.

 a| 
부울

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "id": 3411,
  "method": "GetLoginBanner",
  "params": {}
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
  "id": 3411,
  "result": {
    "loginBanner": {
      "banner": "Welcome to NetApp!",
      "enabled": false
    }
  }
}
----
