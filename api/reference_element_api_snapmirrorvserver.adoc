= 스냅미러V서버
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
snapMirrorVserver 개체에는 대상 ONTAP 클러스터의 스토리지 가상 머신(또는 Vserver)에 대한 정보가 포함되어 있습니다.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
스냅미러엔드포인트ID
 a| 
대상 ONTAP 시스템의 ID입니다.
 a| 
정수



 a| 
vserverName
 a| 
Vserver의 이름입니다.
 a| 
끈



 a| 
vserver 유형
 a| 
Vserver의 유형입니다. 가능한 값:

* 데이터
* 관리자
* 체계
* 마디

 a| 
끈



 a| 
vserver하위 유형
 a| 
Vserver의 하위 유형입니다. 가능한 값:

* 기본
* dp_목적지
* 데이터
* 동기화_소스
* 동기화 대상

 a| 
끈



 a| 
루트볼륨
 a| 
Vserver의 루트 볼륨입니다.
 a| 
끈



 a| 
루트볼륨집계
 a| 
루트 볼륨이 생성될 집계입니다.
 a| 
끈



 a| 
vserverAggregateInfo
 a| 
snapMirrorVserverAggregateInfo 객체의 배열입니다.
 a| 
JSON 객체



 a| 
관리자 상태
 a| 
Vserver의 자세한 관리 상태입니다. 가능한 값:

* 달리기
* 멈췄다
* 시작하다
* 멎는
* 초기화 중
* 삭제

 a| 
끈



 a| 
운영 상태
 a| 
Vserver의 기본 작동 상태. 가능한 값:

* 달리기
* 멈췄다

 a| 
끈

|===
