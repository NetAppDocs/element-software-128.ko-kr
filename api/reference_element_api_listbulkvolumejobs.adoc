= ListBulkVolumeJobs
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListBulkVolumeJobs` 시스템에서 발생하는 각 대량 볼륨 읽기 또는 쓰기 작업에 대한 정보를 얻는 방법입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
대량 볼륨 작업
 a| 
각 대량 작업에 대한 정보 배열입니다.
 a| 
xref:reference_element_api_bulkvolumejob.adoc[대량 볼륨 작업]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "ListBulkVolumeJobs",
   "params": {
      },
   "id" : 1
}
----
