= API 인증
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
API를 사용할 때 모든 API 요청에 HTTP 기본 인증 헤더를 포함하면 시스템에 인증할 수 있습니다.  인증 정보를 생략하면 시스템은 HTTP 401 응답으로 인증되지 않은 요청을 거부합니다.  이 시스템은 TLS를 통한 HTTP 기본 인증을 지원합니다.

API 인증을 위해 클러스터 관리자 계정을 사용하세요.
