= 클러스터 상태 가져오기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetClusterState` 노드가 클러스터의 일부인지 아닌지를 나타내는 API 메서드입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
무리
 a| 
클러스터의 이름.
 a| 
끈



 a| 
상태
 a| 
* 사용 가능: 노드가 클러스터 이름으로 구성되지 않았습니다.
* 보류 중: 노드가 특정 명명된 클러스터에 대해 보류 중이며 추가할 수 있습니다.
* 활성: 노드는 클러스터의 활성 멤버이므로 다른 클러스터에 추가될 수 없습니다.

 a| 
끈

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "GetClusterState",
   "params": {},
   "id" : 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
   "id" : 1,
   "result" :
      "cluster" :  "Cluster101"
      "state" : "Active"
}
----
