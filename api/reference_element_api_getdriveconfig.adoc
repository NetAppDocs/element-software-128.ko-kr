= 드라이브 구성 가져오기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetDriveConfig` 예상되는 슬라이스 및 블록 드라이브 수와 현재 노드에 연결된 슬라이스 및 블록 드라이브 수에 대한 드라이브 정보를 가져오는 방법입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
드라이브 구성
 a| 
노드에 연결된 드라이브에 대한 정보입니다.
 a| 
xref:reference_element_api_drive.adoc[운전하다]

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "GetDriveConfig",
   "params": {},
   "id" : 1
}
----
