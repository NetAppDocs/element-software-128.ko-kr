= CreateSnapMirrorEndpoint
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element 웹 UI는 다음을 사용합니다. `CreateSnapMirrorEndpoint` 원격 SnapMirror 엔드포인트와 관계를 생성하는 방법입니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
관리IP
 a| 
원격 SnapMirror 엔드포인트의 관리 IP 주소입니다.
 a| 
끈
 a| 
None
 a| 
예



 a| 
사용자 이름
 a| 
ONTAP 시스템의 관리 사용자 이름입니다.
 a| 
끈
 a| 
None
 a| 
예



 a| 
비밀번호
 a| 
ONTAP 시스템의 관리 비밀번호입니다.
 a| 
끈
 a| 
None
 a| 
예

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
스냅미러엔드포인트
 a| 
새로 생성된 SnapMirror 엔드포인트.
 a| 
xref:reference_element_api_snapmirrorendpoint.adoc[스냅미러엔드포인트]

|===
