= 네트워크(본딩된 인터페이스)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
네트워크(결합된 인터페이스) 개체에는 스토리지 노드의 결합된 네트워크 인터페이스에 대한 구성 정보가 포함되어 있습니다.  당신은 사용할 수 있습니다 `GetConfig` 그리고 `GetNetworkConfig` 저장 노드에 대한 정보를 얻는 방법.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===


| 이름 | 설명 | 유형 


 a| 
주소
 a| 
노드의 이 인터페이스에 할당된 IPv4 주소입니다.
 a| 
끈



 a| 
addressV6
 a| 
노드의 Bond1G 인터페이스에 할당된 IPv6 관리 주소입니다.
 a| 
끈



 a| 
채권 다운딜레이
 a| 
링크 장애가 감지된 후 슬레이브를 비활성화하기 전까지 기다리는 시간(밀리초)입니다.
 a| 
끈



 a| 
bond-fail_over_mac
 a| 
네트워크 인터페이스의 MAC 주소 구성.
 a| 
끈



 a| 
본드-미몬
 a| 
MII 링크 상태에서 링크 장애가 검사되는 빈도(밀리초)입니다.
 a| 
끈



 a| 
본드 모드
 a| 
본딩 모드. 가능한 값:

* ActivePassive(기본값)
* 장백의
* LACP(권장)

 a| 
끈



 a| 
본드-기본_재선택
 a| 
기본 본드 슬레이브가 활성 슬레이브로 선택되는 시기를 지정합니다. 가능한 값:

* 언제나
* 더 나은
* 실패

 a| 
끈



 a| 
노예들
 a| 
본드에 대한 슬레이브 인터페이스 목록입니다.
 a| 
끈



 a| 
채권-lacp_rate
 a| 
Bond Mode가 LACP인 경우, 요율은 다음 중 하나로 변경될 수 있습니다.

* LACP Fast(기본값)
* LACP 느림

 a| 
끈



 a| 
채권 상향 지연
 a| 
링크가 감지된 후 슬레이브를 활성화하기 전까지 기다리는 시간(밀리초)입니다.
 a| 
끈



 a| 
dns-네임서버
 a| 
쉼표나 공백으로 구분된 도메인 이름 서비스에 사용되는 주소 목록입니다.
 a| 
끈



 a| 
DNS 검색
 a| 
DNS 검색 도메인을 공백이나 쉼표로 구분한 목록입니다.
 a| 
끈



 a| 
가족
 a| 
인터페이스가 사용하도록 구성된 주소 패밀리입니다.  현재 IPv4의 경우 "inet"이 지원됩니다.
 a| 
끈



 a| 
게이트웨이
 a| 
로컬 네트워크에서 트래픽을 보내는 데 사용되는 IPv4 라우터 네트워크 주소입니다.
 a| 
끈



 a| 
gatewayV6
 a| 
로컬 Bond1G 네트워크에서 트래픽을 보내는 데 사용되는 IPv6 라우터 네트워크 주소입니다.
 a| 
끈



 a| 
ipV6PrefixLength
 a| 
Bond1G 네트워크의 IPv6 트래픽에 대한 "net" 유형의 정적 경로에 대한 서브넷 접두사 길이입니다.
 a| 
끈



 a| 
맥어드레스
 a| 
네트워크에서 관찰되고 인터페이스에 할당된 실제 MAC 주소입니다.
 a| 
끈



 a| 
macAddressPermanent
 a| 
제조업체가 인터페이스에 할당한 변경 불가능한 MAC 주소입니다.
 a| 
끈



 a| 
방법
 a| 
인터페이스를 구성하는 데 사용되는 방법입니다. 가능한 값:

* 루프백: IPv4 루프백 인터페이스를 정의하는 데 사용됩니다.
* manual: 자동으로 구성되지 않는 인터페이스를 정의하는 데 사용됩니다.
* dhcp: DHCP를 통해 IP 주소를 얻는 데 사용할 수 있습니다.
* 정적: 정적으로 할당된 IPv4 주소로 이더넷 인터페이스를 정의하는 데 사용됩니다.

 a| 
끈



 a| 
엠티유
 a| 
인터페이스가 전송할 수 있는 가장 큰 패킷 크기(바이트).  1500 이상이어야 하며, 최대 9000까지 지원됩니다.
 a| 
끈



 a| 
넷마스크
 a| 
인터페이스의 서브넷을 지정하는 비트 마스크입니다.
 a| 
끈



 a| 
회로망
 a| 
넷마스크를 기준으로 IP 주소 범위가 시작되는 위치를 나타냅니다.
 a| 
끈



 a| 
경로
 a| 
라우팅 테이블에 적용할 경로 문자열의 쉼표로 구분된 배열입니다.
 a| 
문자열 배열



 a| 
상태
 a| 
인터페이스의 상태. 가능한 값:

* 아래: 인터페이스가 비활성화되었습니다.
* 위쪽: 인터페이스는 준비되었지만 링크가 없습니다.
* UpAndRunning: 인터페이스가 준비되었고 링크가 설정되었습니다.

 a| 
끈



 a| 
대칭 경로 규칙
 a| 
노드에 구성된 대칭 라우팅 규칙입니다.
 a| 
문자열 배열



 a| 
업앤러닝
 a| 
인터페이스가 준비되었고 링크가 있는지 여부를 나타냅니다.
 a| 
부울



 a| 
가상네트워크태그
 a| 
인터페이스의 가상 네트워크 식별자(VLAN 태그).
 a| 
끈

|===
