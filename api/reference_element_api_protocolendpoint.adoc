= 프로토콜엔드포인트
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
protocolEndpoint 객체는 프로토콜 엔드포인트의 속성을 포함합니다.  클러스터의 모든 프로토콜 엔드포인트에 대한 이 정보는 다음을 사용하여 검색할 수 있습니다. `ListProtocolEndpoints` API 방식.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
기본 공급자 ID
 a| 
프로토콜 엔드포인트에 대한 기본 프로토콜 엔드포인트 공급자 개체의 ID입니다.
 a| 
정수



 a| 
프로토콜 엔드포인트 ID
 a| 
프로토콜 엔드포인트의 고유 ID입니다.
 a| 
UUID



 a| 
프로토콜 엔드포인트 상태
 a| 
프로토콜 엔드포인트의 상태입니다. 가능한 값:

* 활성: 프로토콜 엔드포인트가 사용 중입니다.
* 시작: 프로토콜 엔드포인트가 시작됩니다.
* 장애 조치: 프로토콜 엔드포인트가 장애 조치되었습니다.
* 예약됨: 프로토콜 엔드포인트가 예약되었습니다.

 a| 
끈



 a| 
공급자 유형
 a| 
프로토콜 엔드포인트 공급자의 유형입니다. 가능한 값:

* 주요한
* 반성

 a| 
끈



 a| 
scsiNAADeviceID
 a| 
NAA IEEE 등록 확장 형식의 프로토콜 엔드포인트에 대한 전역적으로 고유한 SCSI 장치 식별자입니다.
 a| 
끈



 a| 
보조공급자ID
 a| 
프로토콜 엔드포인트에 대한 보조 프로토콜 엔드포인트 공급자 개체의 ID입니다.
 a| 
정수

|===
