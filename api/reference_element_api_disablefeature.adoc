= 기능 비활성화
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `DisableFeature` 비활성화하는 방법 `VolumeLoadBalanceOnActualIOPS` 특징. `VolumeLoadBalanceOnActualIOPS` 기본적으로 비활성화되어 있습니다.

[NOTE]
====
VVols와 같은 다른 기능을 비활성화하려고 하면 다음과 같은 오류 메시지가 나타납니다.

[listing]
----
{"error":{"code":500,"message":"VolumeLoadBalanceOnActualIOPS is the only feature that can be disabled","name":"xAPINotPermitted"},"id":null}
----
====


== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

[cols="1a,3a,1a,1a,1a"]
|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
특징
 a| 
클러스터 기능을 비활성화합니다.  가능한 값:

* `VolumeLoadBalanceOnActualIOPS`: 최소 IOPS가 아닌 실제 IOPS에 대한 슬라이스 밸런싱을 비활성화합니다.

 a| 
끈
 a| 
None
 a| 
예

|===


== 반환 값

이 메서드에는 반환 값이 없습니다.



== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "DisableFeature",
    "params": {
        "feature" : "VolumeLoadBalanceOnActualIOPS"
    },
    "id": 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
  "id": 1,
  "result": {}
}
----
