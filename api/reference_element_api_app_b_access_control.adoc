= 접근 제어
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
사용 가능한 Element API 메서드는 설정한 액세스 유형에 따라 달라집니다.


NOTE: Element UI에서는 사용할 수 없는 API를 통해 사용 가능한 액세스 유형이 있습니다.



== 계정

계정 액세스 유형에는 다음 방법을 사용할 수 있습니다.

|===


 a|

 a|

 a| 
계정 효율성 얻기



 a| 
계정 삭제

|===
