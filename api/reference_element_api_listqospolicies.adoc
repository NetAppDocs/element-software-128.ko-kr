= ListQoSPolicies
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListQoSPolicies` 시스템의 모든 QoS 정책 설정을 나열하는 방법입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
qos정책
 a| 
각 QoS 정책에 대한 세부 정보 목록입니다.
 a| 
xref:reference_element_api_qospolicy.adoc[QoSPolicy]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "id": 231,
  "method": "ListQoSPolicies",
  "params": {}
}
----
