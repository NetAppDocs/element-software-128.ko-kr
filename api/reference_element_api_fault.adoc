= 잘못
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
오류 개체에는 클러스터에서 감지된 오류에 대한 정보가 포함되어 있습니다.  그만큼 `ListClusterFaults` 이 메서드는 클러스터 오류 정보를 반환합니다.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
블록 업그레이드
 a| 
오류로 인해 업그레이드가 차단되었습니다. 가능한 값:

* true: 오류로 인해 업그레이드가 차단됩니다.
* false: 오류로 인해 업그레이드가 차단되지 않습니다.

 a| 
부울



 a| 
클러스터 오류 ID
 a| 
각 클러스터 오류와 관련된 고유 ID입니다.
 a| 
정수



 a| 
암호
 a| 
감지된 특정 오류에 대한 오류 코드입니다.  자세한 내용은 클러스터 오류 코드를 참조하세요.
 a| 
끈



 a| 
데이터
 a| 
추가적인 오류 관련 정보입니다.
 a| 
JSON 객체



 a| 
날짜
 a| 
UTC+0 형식의 현재 시간입니다.
 a| 
ISO 8601 문자열



 a| 
세부
 a| 
추가 세부 정보가 포함된 오류 설명입니다.
 a| 
끈



 a| 
드라이브ID
 a| 
driveIDs 목록의 첫 번째 드라이브 ID입니다.  driveIDs 목록이 비어 있는 경우(즉, 드라이브를 처리하는 오류가 반환되지 않은 경우), 이 값은 0입니다.
 a| 
정수



 a| 
드라이브 ID
 a| 
이 오류가 참조하는 드라이브의 driveID 값 목록입니다.  드라이브와 관련된 오류에 포함됩니다.  없으면 빈 배열입니다.
 a| 
정수 배열



 a| 
노드 하드웨어 오류 ID
 a| 
클러스터의 하드웨어 오류에 할당된 식별자입니다.
 a| 
정수



 a| 
노드ID
 a| 
이 오류가 참조하는 노드의 노드 ID입니다.  노드 및 드라이브 오류에 포함되고, 그렇지 않은 경우 0으로 설정됩니다.
 a| 
정수



 a| 
해결됨
 a| 
오류의 해결 상태입니다. 가능한 값:

* true: 오류가 더 이상 감지되지 않습니다.
* 거짓: 오류가 여전히 존재합니다.

 a| 
부울



 a| 
해결된 날짜
 a| 
오류가 해결된 날짜와 시간입니다.
 a| 
ISO 8601 문자열



 a| 
서비스ID
 a| 
오류와 관련된 서비스입니다.  오류가 서비스와 연관되지 않은 경우 이 값은 "0"(영)입니다.
 a| 
정수



 a| 
심각성
 a| 
결함의 심각성. 가능한 값:

* 경고: 사소한 문제입니다.  클러스터가 작동하고 있으며 이 심각도 수준에서는 업그레이드가 허용됩니다.
* 오류: 일반적으로 서비스에 영향을 미치지 않는 장애(성능 저하나 HA 손실 가능성 제외).  일부 기능이 비활성화될 수 있습니다.
* 중대한: 서비스에 영향을 미치는 심각한 실패.  시스템이 API 요청이나 클라이언트 I/O를 처리할 수 없어 데이터 손실 위험이 있습니다.
* bestPractice: 최적이 아닌 시스템 구성으로 인해 발생하는 오류.

 a| 
끈



 a| 
유형
 a| 
단층의 종류. 가능한 값:

* 노드: 전체 노드에 영향을 미치는 오류.
* 드라이브: 개별 드라이브에 영향을 미치는 오류.
* 클러스터: 전체 클러스터에 영향을 미치는 오류.
* 서비스: 클러스터의 서비스에 영향을 미치는 오류입니다.
* 볼륨: 개별 볼륨에 영향을 미치는 오류.

 a| 
끈

|===
