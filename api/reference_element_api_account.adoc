= 계정
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
그만큼 `account` 객체에는 계정에 대한 정보가 들어 있습니다.  이 개체에는 계정에 대한 "구성된" 정보만 포함되며 런타임이나 사용 정보는 포함되지 않습니다.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
`accountID`
 a| 
계정의 고유한 계정 ID입니다.
 a| 
정수



 a| 
`attributes`
 a| 
JSON 객체 형식의 이름-값 쌍 목록입니다.
 a| 
JSON 객체



 a| 
`enableChap`
 a| 
CHAP 계정 자격 증명을 초기자가 볼륨에 액세스하는 데 사용할 수 있는지 여부를 지정합니다.
 a| 
부울



 a| 
`initiatorSecret`
 a| 
개시자 CHAP 비밀.
 a| 
끈



 a| 
`status`
 a| 
계정의 현재 상태입니다. 가능한 값:

* 활성: 활성 계정.
* 잠김: 잠긴 계정.
* 제거됨: 삭제 및 정리된 계정입니다.

 a| 
끈



 a| 
`storageContainerID`
 a| 
이 계정과 연결된 가상 볼륨 스토리지 컨테이너의 고유 ID입니다.
 a| 
UUID



 a| 
`targetSecret`
 a| 
대상 CHAP 비밀번호.
 a| 
끈



 a| 
`username`
 a| 
계정의 사용자 이름입니다.
 a| 
끈



 a| 
`volumes`
 a| 
이 계정이 소유한 볼륨의 볼륨 ID 목록입니다.
 a| 
정수 배열

|===
