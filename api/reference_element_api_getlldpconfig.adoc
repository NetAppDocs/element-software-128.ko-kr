= GetLldpConfig
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetLldpConfig` 스토리지 클러스터의 각 노드에 대한 LLDP(Link Layer Discovery Protocol) 구성을 가져오는 방법입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
lldp구성
 a| 
스토리지 클러스터 LLDP 구성에 대한 정보입니다.
 a| 
JSON 객체

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "GetLldpConfig",
   "id" : 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
	"id": null,
		"result": {
			"lldpConfig": {
				"enableLldp": true,
				"enableMed": false,
				"enableOtherProtocols": true
		}
	}
}
----
