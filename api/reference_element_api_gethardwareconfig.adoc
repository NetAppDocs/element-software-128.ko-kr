= 하드웨어 구성 가져오기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetHardwareConfig` 노드에 대한 하드웨어 구성 정보를 가져오는 방법입니다.  이 구성 데이터는 내부용으로 사용됩니다.  더욱 유용한 라이브 시스템 하드웨어 구성 요소 인벤토리를 얻으려면 다음을 사용하세요. `GetHardwareInfo` 대신 방법을 사용하세요.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


| 하드웨어 구성 | 하드웨어 정보 및 현재 설정 목록입니다. | JSON 객체 
|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "GetHardwareConfig",
   "params": {},
   "id" : 1
}
----
