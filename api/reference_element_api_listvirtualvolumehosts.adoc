= 가상 볼륨 호스트 목록
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListVirtualVolumeHosts` 클러스터에 알려진 모든 가상 볼륨 호스트 목록을 가져오는 방법입니다.  가상 볼륨 호스트는 VASA API 공급자와 세션을 시작한 VMware ESX 호스트입니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
가상 볼륨 호스트 ID
 a| 
정보를 검색할 가상 볼륨 호스트 ID 목록입니다.  이 매개변수를 생략하면 메서드는 모든 가상 볼륨 호스트에 대한 정보를 반환합니다.
 a| 
virtualVolumeHostID UUID 배열
 a| 
None
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
호스트
 a| 
클러스터의 가상 볼륨 호스트를 설명하는 개체 목록입니다.
 a| 
xref:reference_element_api_host.adoc[주인]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "ListVirtualVolumeHosts",
    "params": {
    },
    "id": 1
}
----
