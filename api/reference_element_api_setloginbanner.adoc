= 로그인 배너 설정
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `SetLoginBanner` 사용자가 Element 웹 인터페이스에 로그인할 때 표시되는 이용 약관 배너를 구성하는 방법입니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
기치
 a| 
이용 약관 배너에 원하는 텍스트입니다.  허용되는 최대 길이는 4,096자입니다.
 a| 
끈
 a| 
None
 a| 
아니요



 a| 
활성화됨
 a| 
이용 약관 배너의 상태입니다. 가능한 값:

* `true`: 이용 약관 배너는 웹 인터페이스 로그인 시 표시됩니다.
* `false`: 웹 인터페이스 로그인 시 이용 약관 배너가 표시되지 않습니다.

 a| 
부울
 a| 
None
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
기치
 a| 
이용 약관 배너의 현재 텍스트입니다.  배너가 비활성화된 경우에도 이 값에는 텍스트가 포함될 수 있습니다.
 a| 
끈



 a| 
활성화됨
 a| 
이용 약관 배너의 상태입니다. 가능한 값:

* true: 웹 인터페이스 로그인 시 이용 약관 배너가 표시됩니다.
* false: 웹 인터페이스 로그인 시 이용 약관 배너가 표시되지 않습니다.

 a| 
부울

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "id": 3920,
  "method": "SetLoginBanner",
  "params": {
    "banner": "Welcome to NetApp!",
    "enabled": true
  }
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
  "id": 3920,
  "result": {
    "loginBanner": {
      "banner": "Welcome to NetApp!",
      "enabled": true
    }
  }
}
----
