= 스냅미러 관계 중단
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element 소프트웨어 웹 UI는 다음을 사용합니다. `AbortSnapMirrorRelationship` 시작되었지만 아직 완료되지 않은 SnapMirror 전송을 중지하는 방법입니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
스냅미러엔드포인트ID
 a| 
Element 스토리지 클러스터와 통신하는 원격 ONTAP 스토리지 시스템의 엔드포인트 ID입니다.
 a| 
정수
 a| 
None
 a| 
예



 a| 
목적지볼륨
 a| 
SnapMirror 관계의 대상 볼륨입니다.
 a| 
xref:reference_element_api_snapmirrorvolumeinfo.adoc[스냅미러볼륨정보]
 a| 
None
 a| 
예



 a| 
클리어체크포인트
 a| 
재시작 체크포인트를 지울지 여부를 결정합니다. 가능한 값:

* true
* 거짓

 a| 
부울
 a| 
거짓
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
스냅미러관계
 a| 
중단된 SnapMirror 관계에 대한 정보를 담고 있는 객체입니다.
 a| 
xref:reference_element_api_snapmirrorrelationship.adoc[스냅미러관계]

|===
