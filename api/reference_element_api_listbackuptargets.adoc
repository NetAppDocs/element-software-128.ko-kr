= 백업 대상 목록
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListBackupTargets` 생성된 모든 백업 대상에 대한 정보를 가져오는 방법입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
백업 대상
 a| 
각 백업 대상에 대해 반환된 객체입니다.  포함된 객체:

* 속성: JSON 객체 형식의 이름-값 쌍 목록입니다.  (JSON 객체)
* backupTargetID: 백업 대상에 할당된 고유 식별자입니다.  (정수)
* name: 백업 대상의 이름입니다.  (끈)

 a| 
JSON 객체

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "ListBackupTargets",
   "params": {},
   "id": 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
   "id": 1,
   "result": {
     "backupTargets": [
       {
        "attributes" : {},
        "backupTargetID" : 1,
        "name" : "mytargetbackup"
       }
    ]
  }
}
----
