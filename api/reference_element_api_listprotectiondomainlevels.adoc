= ListProtectionDomainLevels
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListProtectionDomainLevels` 스토리지 클러스터의 허용 범위와 복원력 수준을 나열하는 방법입니다.  허용 수준은 장애 발생 시에도 클러스터가 데이터를 계속 읽고 쓸 수 있는 능력을 나타내고, 복원력 수준은 스토리지 클러스터가 하나 이상의 장애로부터 자동으로 복구할 수 있는 능력을 나타냅니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
보호 도메인 수준
 a| 
각 보호 도메인 수준이 스토리지 클러스터의 허용 범위와 복원력 정보를 제공하는 목록입니다.
 a| 
xref:reference_element_api_protectiondomainlevel.adoc[보호 도메인 수준]

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "ListProtectionDomainLevels",
  "params": {},
  "id" : 1
}
----
