= 원격 클러스터 스냅샷 상태
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
그만큼 `remoteClusterSnapshotStatus` 개체에는 원격 스토리지 클러스터에 저장된 스냅샷의 UUID와 상태가 포함되어 있습니다.  이 정보는 다음을 통해 얻을 수 있습니다. `ListSnapshots` 또는 `ListGroupSnapshots` API 방법.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
원격 상태
 a| 
소스 클러스터에서 본 대상 클러스터의 원격 스냅샷 복제 상태입니다. 가능한 값:

* 현재: 스냅샷이 원격 클러스터에 있습니다.
* NotPresent: 스냅샷이 원격 클러스터에 존재하지 않습니다.
* 동기화: 이는 대상 클러스터이며 현재 스냅샷을 복제하고 있습니다.
* 삭제됨: 이것은 대상 클러스터입니다.  스냅샷은 삭제되었지만 소스에는 여전히 남아 있습니다.

 a| 
끈



 a| 
볼륨페어UUID
 a| 
볼륨 쌍의 범용 식별자입니다.
 a| 
UUID

|===
