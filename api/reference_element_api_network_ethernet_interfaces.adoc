= 네트워크(이더넷 인터페이스)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
네트워크(이더넷 인터페이스) 개체에는 개별 이더넷 인터페이스에 대한 구성 정보가 포함되어 있습니다.  당신은 사용할 수 있습니다 `GetConfig` 그리고 `GetNetworkConfig` 저장 노드에 대한 정보를 얻는 방법.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===


| 이름 | 설명 | 유형 


 a| 
본드 마스터
 a| 
이 물리적 인터페이스가 본드 슬레이브로 결합된 본드 인터페이스를 지정합니다.
 a| 
끈



 a| 
가족
 a| 
인터페이스가 사용하도록 구성된 주소 패밀리입니다.  현재 IPv4의 경우 "inet"이 지원됩니다.
 a| 
끈



 a| 
맥어드레스
 a| 
네트워크에서 관찰되고 인터페이스에 할당된 실제 MAC 주소입니다.
 a| 
끈



 a| 
macAddressPermanent
 a| 
제조업체가 인터페이스에 할당한 변경 불가능한 MAC 주소입니다.
 a| 
끈



 a| 
방법
 a| 
인터페이스를 구성하는 데 사용되는 방법입니다. 가능한 값:

* 루프백: IPv4 루프백 인터페이스를 정의하는 데 사용됩니다.
* manual: 자동으로 구성되지 않는 인터페이스를 정의하는 데 사용됩니다.
* dhcp: DHCP를 통해 IP 주소를 얻는 데 사용할 수 있습니다.
* 정적: 정적으로 할당된 IPv4 주소로 이더넷 인터페이스를 정의하는 데 사용됩니다.

 a| 
끈



 a| 
상태
 a| 
인터페이스의 상태. 가능한 값:

* 아래: 인터페이스가 비활성화되었습니다.
* 위쪽: 인터페이스는 준비되었지만 링크가 없습니다.
* UpAndRunning: 인터페이스가 준비되었고 링크가 설정되었습니다.

 a| 
끈



 a| 
업앤러닝
 a| 
인터페이스가 준비되었고 링크가 있는지 여부를 나타냅니다.
 a| 
부울

|===
