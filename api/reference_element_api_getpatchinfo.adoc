= 패치 정보 가져오기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetPatchInfo` 스토리지 노드에 설치된 Element 소프트웨어 패치에 대한 정보를 얻는 방법입니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


| `force`  a| 
스토리지 클러스터의 모든 노드에서 메서드를 강제로 실행합니다.  단일 노드 대신 클러스터 IP 주소에 API를 발급하는 경우에만 이 기능이 필요합니다. 가능한 값:

* `true`
* `false`

| 부울 | `false` | 아니요 
|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


| 패치 | 이 노드에 설치된 패치에 대한 정보를 담고 있는 객체입니다. | JSON 객체 
|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "GetPatchInfo",
   "params": {
      "force": false,
      },
   "id": 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
    "id": 1,
    "result": {
        "patches": {
          "SUST936": {
          "date": "Wed 09 Dec 2020 10:41:59 PM UTC",
          "description": "BMC fixes",
          "newFiles": [
              "None"
          ],
          "patchedFiles": [
              "Patched_file_1.bin",
              "Patched_file_2.dat",
              "Patched_file_3.tgz"
          ]
          }
        }
    }
}

----
