= 로그인 세션 정보 가져오기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetLoginSessionInfo` 로그인 셸과 TUI 모두에 대해 로그인 인증 세션이 유효한 기간을 반환하는 방법입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
로그인세션정보
 a| 
인증 만료 기간을 포함하는 객체입니다.  반환될 수 있는 객체:

* 시간 초과:
+
세션이 시간 초과되어 만료되는 시간(분)입니다.  H:mm:ss 형식으로 지정되었습니다.  예를 들어: 1:30:00, 20:00, 5:00.  타임아웃이 입력된 형식에 관계없이 모든 선행 0과 콜론이 제거됩니다.


 a| 
JSON 객체

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "GetLoginSessionInfo",
    "params": {},
  "id" : 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
"id": 1,
  "result" : {
    "loginSessionInfo" : {
      "timeout" : "30:00"
    }
  }
}
----
