= CreateQoSPolicy
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `CreateQoSPolicy` 나중에 볼륨을 생성하거나 수정할 때 적용할 수 있는 QoSPolicy 객체를 만드는 방법입니다.  QoS 정책에는 고유한 ID, 이름, QoS 설정이 있습니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
이름
 a| 
QoS 정책의 이름입니다(예: 골드, 플래티넘 또는 실버).
 a| 
끈
 a| 
None
 a| 
예



 a| 
qos
 a| 
이 정책이 나타내는 QoS 설정입니다.
 a| 
xref:reference_element_api_qos.adoc[서비스 품질]
 a| 
None
 a| 
예

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
qos정책
 a| 
새로 생성된 QoSPolicy 객체.
 a| 
xref:reference_element_api_qospolicy.adoc[QoSPolicy]

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "id": 68,
  "method": "CreateQoSPolicy",
  "params": {
    "name": "bronze",
    "qos": {
      "minIOPS": 50,
      "maxIOPS": 15000,
      "burstIOPS": 15000
    }
  }
}
----
