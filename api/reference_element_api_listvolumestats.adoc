= 볼륨 통계 목록
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListVolumeStats` 단일 볼륨, 볼륨 목록 또는 모든 볼륨(volumeIDs 매개변수를 생략한 경우)에 대한 높은 수준의 활동 측정값을 얻는 방법입니다.  측정값은 볼륨 생성부터 누적됩니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
가상 볼륨 포함
 a| 
가상 볼륨은 기본적으로 응답에 포함됩니다.  가상 볼륨을 제외하려면 false로 설정합니다.
 a| 
부울
 a| 
true
 a| 
아니요



 a| 
볼륨 ID
 a| 
활동 정보를 검색할 볼륨 목록입니다.
 a| 
정수 배열
 a| 
아니요
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
볼륨통계
 a| 
볼륨 활동 정보 목록입니다.
 a| 
xref:reference_element_api_volumestats.adoc[볼륨통계]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "ListVolumeStats",
    "params": {
        "volumeIDs": [1]
    },
    "id": 1
}
----
