= 페어링된 클러스터에 지원되는 복제 모드
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
쌍을 이루는 클러스터에서는 다음과 같은 복제 모드가 지원됩니다.

* 데이터의 비동기 복제: 복제 대상 볼륨으로 전송되는 데이터는 비동기적으로 전송됩니다.  시스템은 데이터를 쓰기 전에 확인이 전송될 때까지 기다리지 않습니다.
* 데이터의 동기 복제: 복제 대상 볼륨으로 전송되는 데이터는 동기적으로 전송됩니다.  호스트에서 보낸 I/O 작업이 시스템에서 확인되면 시스템 확인이 호스트로 다시 전송되고 데이터는 복제 대상 볼륨으로 전송됩니다.
* 스냅샷 전용 데이터 복제: 볼륨 스냅샷만 대상 클러스터에 복제됩니다.
