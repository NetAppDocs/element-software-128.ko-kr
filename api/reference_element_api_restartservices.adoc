= 서비스 재시작
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `RestartServices` 노드에서 서비스를 다시 시작하는 방법입니다.



== 매개변수

주의:

이 방법을 사용하면 노드 서비스가 일시적으로 중단됩니다.

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
힘
 a| 
노드에서 서비스를 다시 시작하려면 true로 설정합니다.
 a| 
부울
 a| 
None
 a| 
예



 a| 
서비스
 a| 
다시 시작할 서비스 이름입니다.
 a| 
끈
 a| 
None
 a| 
아니요



 a| 
행동
 a| 
서비스에서 수행할 작업(시작, 중지, 다시 시작).
 a| 
끈
 a| 
None
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
세부
 a| 
오류(있는 경우)를 포함한 서비스 재시작 절차의 출력입니다.
 a| 
JSON 객체



 a| 
지속
 a| 
노드에 대한 서비스를 다시 시작하는 데 걸린 시간(초)입니다.
 a| 
끈



 a| 
결과
 a| 
재시작 결과.
 a| 
끈

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "RestartServices",
   "params": {
              "force" : true
              "action" : restart,
             }
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
  "id": 1,
  "result": {
    "details": "solidfire stop/waiting\nsolidfire start/running, process 7284\n",
    "duration": "00:00:02.541594",
    "result": "Passed"
  }
}
----
