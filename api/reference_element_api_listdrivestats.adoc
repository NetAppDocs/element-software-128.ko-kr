= 드라이브 통계 목록
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListDriveStats` 클러스터 내 여러 드라이브에 대한 높은 수준의 활동 측정값을 나열하는 방법입니다.  기본적으로 이 방법은 클러스터의 모든 드라이브에 대한 통계를 반환하며, 이러한 측정값은 클러스터에 드라이브를 추가한 이후부터 누적됩니다.  이 메서드가 반환하는 일부 값은 블록 드라이브에만 해당하고, 일부는 메타데이터 드라이브에만 해당합니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
드라이브
 a| 
드라이브 통계를 반환할 드라이브 ID(driveID) 목록입니다.  이 매개변수를 생략하면 모든 드라이브의 측정값이 반환됩니다.
 a| 
정수 배열
 a| 
None
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
드라이브 통계
 a| 
각 드라이브에 대한 드라이브 활동 정보 목록입니다.
 a| 
xref:reference_element_api_drivestats.adoc[드라이브 통계]정렬



 a| 
오류
 a| 
이 목록에는 드라이브ID와 관련 오류 메시지가 포함되어 있습니다.  항상 존재하며, 오류가 없으면 비어 있습니다.
 a| 
JSON 객체 배열

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "id": 1,
  "method": "ListDriveStats",
  "params": {
    "drives":[22,23]
  }
}
----
