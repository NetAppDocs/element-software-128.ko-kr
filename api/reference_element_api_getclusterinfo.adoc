= 클러스터 정보 가져오기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetClusterInfo` 클러스터에 대한 구성 정보를 반환하는 방법입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
클러스터정보
 a| 
클러스터 정보.
 a| 
xref:reference_element_api_clusterinfo.adoc[클러스터정보]

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "GetClusterInfo",
   "params": {},
   "id" : 1
}
----
