loggingServer 개체에는 스토리지 클러스터에 대해 구성된 모든 로깅 호스트에 대한 정보가 포함되어 있습니다.  사용할 수 있습니다 `GetRemoteLoggingHosts` 현재 로깅 호스트가 무엇인지 확인한 다음 사용하세요. `SetRemoteLoggingHosts` 현재 및 새로운 로깅 호스트의 원하는 목록을 설정합니다.
