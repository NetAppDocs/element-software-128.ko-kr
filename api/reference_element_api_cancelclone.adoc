= 복제 취소
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `CancelClone` 진행 중인 볼륨 복제 또는 볼륨 복사 프로세스를 중지하는 방법입니다.  그룹 복제 작업을 취소하면 시스템이 작업을 완료하고 해당 작업과 관련된 asyncHandle을 제거합니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
클론ID
 a| 
진행 중인 복제 프로세스의 cloneID입니다.
 a| 
정수
 a| 
None
 a| 
예

|===


== 반환 값

이 메서드에는 반환 값이 없습니다.



== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "CancelClone",
   "params": {
      "cloneID" : 5,
   },
   "id" : 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
  "id" : 1,
  "result" : {}
}
----
