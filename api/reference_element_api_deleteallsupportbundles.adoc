= 모든 지원 번들 삭제
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `DeleteAllSupportBundles` 생성된 모든 지원 번들을 삭제하는 방법 `CreateSupportBundle` API 방식.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드에는 반환 값이 없습니다.



== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "DeleteAllSupportBundles",
   "params": {}
  },
  "id": 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
  "id" : 1,
  "result" : {}
 }
}
----
