= 계정 볼륨 목록
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListVolumesForAccount` 계정에 대한 활성 볼륨과 (보류 중인) 삭제 볼륨을 나열하는 방법입니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
가상 볼륨 포함
 a| 
가상 볼륨은 기본적으로 응답에 포함됩니다.  가상 볼륨을 제외하려면 false로 설정합니다.
 a| 
부울
 a| 
true
 a| 
아니요



 a| 
계정ID
 a| 
이 계정 ID가 소유한 모든 볼륨이 반환됩니다.
 a| 
정수
 a| 
아니요
 a| 
예

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
볼륨
 a| 
볼륨 정보 목록입니다.
 a| 
xref:reference_element_api_volume.adoc[용량]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "ListVolumesForAccount",
   "params": {
      "accountID"  : 1
   },
   "id" : 1
}
----
