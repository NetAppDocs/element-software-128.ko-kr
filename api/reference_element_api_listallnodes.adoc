= 모든 노드 나열
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListAllNodes` 클러스터에서 활성 노드와 보류 중인 노드를 나열하는 방법입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
노드
 a| 
클러스터의 활성 노드를 설명하는 개체 목록입니다.
 a| 
xref:reference_element_api_node.adoc[마디]



 a| 
보류 중인 활성 노드
 a| 
클러스터의 보류 중인 활성 노드를 설명하는 개체 목록입니다.
 a| 
xref:reference_element_api_pendingactivenode.adoc[보류 중인 활성 노드]정렬



 a| 
보류 중인 노드
 a| 
클러스터의 보류 중인 노드를 설명하는 개체 목록입니다.
 a| 
xref:reference_element_api_pendingnode.adoc[보류 중인 노드]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "ListAllNodes",
   "params": {},
   "id" : 1
}
----
