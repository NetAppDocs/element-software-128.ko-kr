= 스냅미러볼륨정보
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
snapMirrorVolumeInfo 개체에는 SnapMirror 관계에서 볼륨 위치에 대한 정보(예: 볼륨의 이름 및 유형)가 포함되어 있습니다.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
유형
 a| 
볼륨의 유형. 가능한 값:

* solidfire: 볼륨은 Element 소프트웨어를 실행하는 스토리지 클러스터에 있습니다.
* ontap: 볼륨이 원격 ONTAP 클러스터에 있습니다.

 a| 
끈



 a| 
볼륨ID
 a| 
볼륨의 ID입니다.  "type"이 solidfire인 경우에만 유효합니다.
 a| 
정수



 a| 
v서버
 a| 
이 볼륨을 소유한 Vserver의 이름입니다.  "type"이 ontap인 경우에만 유효합니다.
 a| 
끈



 a| 
이름
 a| 
볼륨의 이름입니다.
 a| 
끈

|===
