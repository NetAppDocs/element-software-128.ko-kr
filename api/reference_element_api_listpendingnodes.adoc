= ListPendingNodes
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListPendingNodes` 시스템에서 보류 중인 저장 노드를 나열하는 방법입니다.  보류 중인 노드는 스토리지 클러스터에 가입하도록 실행되고 구성되었지만 아직 AddNodes API 메서드를 사용하여 추가되지 않은 스토리지 노드입니다.



== IPv4 및 IPv6 관리 주소

참고하십시오 `ListPendingNodes` 관리 IP 주소(MIP)와 관리 가상 IP 주소(MVIP)에 대해 서로 다른 주소 유형을 갖는 보류 중인 노드는 나열되지 않습니다.  예를 들어 보류 중인 노드에 IPv6 MVIP와 IPv4 MIP가 있는 경우 `ListPendingNodes` 결과에 노드가 포함되지 않습니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
보류 중인 노드
 a| 
클러스터에서 보류 중인 노드 목록입니다.
 a| 
xref:reference_element_api_pendingnode.adoc[보류 중인 노드]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "ListPendingNodes",
   "params": {},
   "id" : 1
}
----
