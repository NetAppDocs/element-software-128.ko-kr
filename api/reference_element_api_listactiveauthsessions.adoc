= ListActiveAuthSessions
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListActiveAuthSessions` 모든 활성 인증 세션을 나열하는 방법입니다.  관리자 액세스 권한이 있는 사용자만 이 메서드를 호출할 수 있습니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
세션
 a| 
인증 세션에 대한 세션 정보입니다.
 a| 
link:reference_element_api_authsessioninfo.html["인증 세션 정보"]

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "ListActiveAuthSessions"
}
----

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{

      {

      }
    ]
}
----
