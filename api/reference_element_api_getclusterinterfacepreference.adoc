= GetClusterInterfacePreference
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
그만큼 `GetClusterInterfacePreference` 이 방법을 사용하면 Element 소프트웨어를 실행하는 스토리지 클러스터와 통합된 시스템에서 기존 클러스터 인터페이스 기본 설정에 대한 정보를 얻을 수 있습니다.  이 방법은 내부적으로 사용됩니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
이름
 a| 
클러스터 인터페이스 기본 설정의 이름입니다.
 a| 
끈
 a| 
None
 a| 
예

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
선호
 a| 
요청된 클러스터 인터페이스 기본 설정의 이름과 값입니다.
 a| 
JSON 객체

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
	"method": "GetClusterInterfacePreference",
	"params": {
		"name": "prefname"
		},
	"id": 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
	"id": 1,
	"result": {
		"preference": {
			"name": "prefname",
			"value": "testvalue"
		}
	}
}
----
