= DeleteAuthSession
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `DeleteAuthSession` 개별 사용자 인증 세션을 삭제하는 방법입니다.  호출하는 사용자가 ClusterAdmins/Administrator AccessGroup에 없으면 호출하는 사용자에게 속한 인증 세션만 삭제할 수 있습니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
세션ID
 a| 
삭제할 인증 세션의 고유 식별자입니다.
 a| 
UUID
 a| 
None
 a| 
예

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
세션
 a| 
인증 세션 삭제에 대한 세션 정보입니다.
 a| 
link:reference_element_api_authsessioninfo.html["인증 세션 정보"]

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
    "method": "DeleteAuthSession",
    "params": {
        "sessionID": "a862a8bb-2c5b-4774-a592-2148e2304713"
    },
    "id": 1
}
----
