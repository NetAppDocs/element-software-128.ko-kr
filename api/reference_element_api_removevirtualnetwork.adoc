= 가상 네트워크 제거
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `RemoveVirtualNetwork` 이전에 추가된 가상 네트워크를 제거하는 방법.


NOTE: 이 방법에는 매개변수로 virtualNetworkID 또는 virtualNetworkTag가 필요하지만 둘 다 필요하지는 않습니다.


NOTE: 가상 네트워크에 연결된 이니시에이터가 있는 경우 해당 가상 네트워크를 제거할 수 없습니다.  먼저 개시자의 연결을 해제한 다음 가상 네트워크를 제거합니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
가상네트워크ID
 a| 
제거할 가상 네트워크를 식별하는 네트워크 ID입니다.
 a| 
정수
 a| 
None
 a| 
예



 a| 
가상네트워크태그
 a| 
제거할 가상 네트워크를 식별하는 네트워크 태그입니다.
 a| 
정수
 a| 
None
 a| 
예

|===


== 반환 값

이 메서드에는 반환 값이 없습니다.



== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "RemoveVirtualNetwork",
  "params": {
     "virtualNetworkID": 5
    }
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
   "id": 1,
   "result": {}
}
----
