= 네트워크 인터페이스
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
networkInterface 개체에는 스토리지 노드의 개별 네트워크 인터페이스에 대한 구성 정보가 포함되어 있습니다.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===


| 이름 | 설명 | 유형 


 a| 
주소
 a| 
인터페이스의 IPv4 관리 주소입니다.
 a| 
끈



 a| 
addressV6
 a| 
인터페이스의 IPv6 관리 주소입니다.
 a| 
끈



 a| 
방송
 a| 
인터페이스의 브로드캐스트 주소.
 a| 
끈



 a| 
맥어드레스
 a| 
인터페이스의 MAC 주소.
 a| 
끈



 a| 
엠티유
 a| 
인터페이스의 최대 전송 단위(바이트)입니다.
 a| 
정수



 a| 
이름
 a| 
인터페이스의 이름.
 a| 
끈



 a| 
네임스페이스
 a| 
이 인터페이스에 가상 네트워크 네임스페이스가 할당되었는지 여부입니다.
 a| 
부울



 a| 
넷마스크
 a| 
인터페이스의 서브넷 마스크.
 a| 
끈



 a| 
상태
 a| 
인터페이스의 작동 상태.
 a| 
끈



 a| 
유형
 a| 
인터페이스 유형(본드 마스터, 본드 슬레이브 등)
 a| 
끈



 a| 
가상네트워크태그
 a| 
가상 네트워크의 인터페이스에 할당된 VLAN ID입니다.
 a| 
정수

|===
