= 리셋노드
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ResetNode` 노드를 공장 설정으로 재설정하는 방법입니다.  이 메서드를 호출하면 모든 데이터, 패키지(소프트웨어 업그레이드 등), 구성 및 로그 파일이 노드에서 삭제됩니다.  하지만 이 작업 중에도 노드의 네트워크 설정은 보존됩니다.  클러스터에 참여하는 노드는 공장 설정으로 재설정할 수 없습니다.



== 매개변수

ResetNode API는 "사용 가능" 상태의 노드에서만 사용할 수 있습니다.  클러스터에서 "활성" 상태인 노드나 "보류" 상태인 노드에서는 사용할 수 없습니다.

주의:

이 방법은 노드에 있는 모든 고객 데이터를 지웁니다.

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

== 반환 값

이 메서드에는 반환 값이 없습니다.



== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "ResetNode",
   "params": {
      "build" : "file:///sf/rtfi/image/filesystem.squashfs",
      "force" : true
              },
   "id" : 1
}
----
