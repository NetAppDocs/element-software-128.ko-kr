= snmpTrapRecipient
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
snmpTrapRecipient 개체에는 스토리지 클러스터에서 생성된 SNMP 트랩을 수신하도록 구성된 호스트에 대한 정보가 포함되어 있습니다.  당신은 사용할 수 있습니다 `GetSnmpTrapInfo` SNMP 트랩을 수신하도록 구성된 호스트 목록을 가져오는 API 메서드입니다.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
주인
 a| 
대상 호스트의 IP 주소 또는 호스트 이름입니다.
 a| 
끈



 a| 
포트
 a| 
트랩을 보내야 하는 호스트의 UDP 포트 번호입니다.  유효 범위는 1~65535입니다.  0(영)은 유효한 포트 번호가 아닙니다. 기본 포트는 162입니다.
 a| 
정수



 a| 
지역 사회
 a| 
SNMP 커뮤니티 문자열.
 a| 
끈

|===
