= 네트워킹 재시작
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `RestartNetworking` 노드에서 네트워크 서비스를 다시 시작하는 방법입니다.

주의:

이 방법을 사용하면 노드의 모든 네트워크 서비스가 다시 시작되어 네트워크 연결이 일시적으로 끊어집니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
힘
 a| 
노드에서 네트워크 서비스를 다시 시작하려면 true로 설정합니다.
 a| 
부울
 a| 
None
 a| 
예

|===


== 반환 값

이 메서드에는 반환 값이 없습니다.



== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "RestartNetworking",
   "params": {
              "force" : true
             },
   "id" : 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{ "id" : 1,
  "result" : {}
}
----
