= EnableBmcColdReset
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `EnableBmcColdReset` 클러스터의 모든 노드에 대해 Baseboard Management Controller(BMC)를 주기적으로 재설정하는 백그라운드 작업을 활성화하는 방법입니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
타임아웃
 a| 
BMC 재설정 작업 사이의 시간(분)입니다.
 a| 
정수
 a| 
20160분
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
cBmcResetDurationMinutes
 a| 
재설정 간격 사이의 시간을 반환합니다.  명령이 완료된 후에는 간격이 항상 0이 되어야 합니다.
 a| 
정수

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
    "method": "EnableBmcColdReset",
    "params": {
                "timeout": 36000
    },
    "id": 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
    "id": 1,
    "result": {
        "cBmcResetDurationMinutes": 36000
    }
}
----
