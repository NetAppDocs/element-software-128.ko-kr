[role="lead"]
snapMirrorClusterIdentity 개체에는 SnapMirror 관계에 있는 원격 ONTAP 클러스터에 대한 식별 정보가 포함되어 있습니다.



== 객체 멤버

이 개체에는 다음과 같은 멤버가 포함되어 있습니다.

|===
| 이름 | 설명 | 유형 


 a| 
스냅미러엔드포인트ID
 a| 
대상 ONTAP 시스템의 ID입니다.
 a| 
정수



 a| 
클러스터 이름
 a| 
대상 ONTAP 클러스터의 이름입니다.
 a| 
끈



 a| 
클러스터UUID
 a| 
대상 ONTAP 클러스터의 128비트 범용 고유 식별자입니다.
 a| 
끈



 a| 
클러스터 일련번호
 a| 
대상 ONTAP 클러스터의 일련 번호입니다.
 a| 
끈

|===
