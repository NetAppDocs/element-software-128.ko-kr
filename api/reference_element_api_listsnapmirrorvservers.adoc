= ListSnapMirrorVservers
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element 소프트웨어 웹 UI는 다음을 사용합니다. `ListSnapMirrorVservers` 원격 ONTAP 시스템에서 사용 가능한 모든 SnapMirror Vserver를 나열하는 방법입니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
스냅미러엔드포인트ID
 a| 
지정된 엔드포인트 ID와 연결된 Vserver만 나열합니다.  엔드포인트 ID가 제공되지 않으면 시스템은 알려진 모든 SnapMirror 엔드포인트의 Vserver를 나열합니다.
 a| 
정수
 a| 
None
 a| 
아니요



 a| 
vserver 유형
 a| 
지정된 유형의 Vserver만 나열합니다. 가능한 값:

* 관리자
* 데이터
* 마디
* 체계

 a| 
끈
 a| 
None
 a| 
아니요



 a| 
vserverName
 a| 
지정된 이름을 가진 Vserver만 나열합니다.
 a| 
끈
 a| 
None
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
스냅미러V서버
 a| 
ONTAP 스토리지 시스템에서 사용 가능한 SnapMirror Vserver 목록입니다.
 a| 
xref:reference_element_api_snapmirrorvserver.adoc[스냅미러V서버]정렬

|===
