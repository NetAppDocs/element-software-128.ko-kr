= 리셋드라이브
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ResetDrives` 드라이브를 사전에 초기화하고 현재 드라이브에 있는 모든 데이터를 제거하는 방법입니다.  그런 다음 드라이브를 기존 노드에서 재사용하거나 업그레이드된 노드에서 사용할 수 있습니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
드라이브
 a| 
재설정할 장치 이름 목록(드라이브 ID 아님)입니다.
 a| 
끈
 a| 
None
 a| 
예



 a| 
힘
 a| 
드라이브를 재설정하려면 true로 설정합니다.
 a| 
부울
 a| 
None
 a| 
예

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
세부
 a| 
재설정되는 드라이브의 세부 정보입니다.
 a| 
JSON 객체 배열

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "ResetDrives",
   "params": {
      "drives" : "slot3",
      "force" : true
   },
   "id" : 1
}
----
