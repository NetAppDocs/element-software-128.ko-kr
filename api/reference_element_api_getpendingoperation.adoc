= 보류 중인 작업 가져오기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetPendingOperation` 현재 진행 중인 노드에서 작업을 감지하는 방법입니다.  이 방법은 작업이 완료된 시점을 보고하는 데에도 사용할 수 있습니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
보류 중
 a| 
가능한 값:

* true: 작업이 아직 진행 중입니다.
* 거짓:
+
해당 작업은 더 이상 진행되지 않습니다.


 a| 
부울



 a| 
작업
 a| 
진행 중이거나 완료된 작업의 이름입니다.
 a| 
끈

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "GetPendingOperation",
   "params": {},
   "id" : 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
   "id" : 1,
   "result" : {
      "pendingOperation" : {
         "pending" : "true",
         "operation" : "TestDrivesInternal",
      }
   }
}
----
