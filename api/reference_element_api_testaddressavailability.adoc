= 테스트 주소 가용성
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `TestAddressAvailability` 스토리지 클러스터 내의 인터페이스에서 특정 IP 주소가 사용 중인지 확인하는 방법입니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
인터페이스
 a| 
대상 네트워크 인터페이스(예: eth0, Bond10G 등)
 a| 
끈
 a| 
None
 a| 
예



 a| 
주소
 a| 
대상 인터페이스에서 스캔할 IP 주소입니다.
 a| 
끈
 a| 
None
 a| 
예



 a| 
가상네트워크태그
 a| 
대상 VLAN ID입니다.
 a| 
정수
 a| 
None
 a| 
아니요



 a| 
타임아웃
 a| 
대상 주소를 테스트하는 데 걸리는 시간(초)입니다.
 a| 
정수
 a| 
5
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
주소
 a| 
테스트된 IP 주소입니다.
 a| 
끈



 a| 
사용 가능
 a| 
요청한 IP 주소가 사용 중이면 True이고, 그렇지 않으면 False입니다.
 a| 
부울

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
	"method": "TestAddressAvailability",
	"params": {
		"interface": "Bond10G",
		"address": "10.0.0.1",
		"virtualNetworkTag": 1234
	}
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
	"id": 1,
	"result": {
		"address":"10.0.0.1",
		"available": true
	}
}
----
