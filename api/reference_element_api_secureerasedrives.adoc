= 보안 삭제 드라이브
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `SecureEraseDrives` "사용 가능" 상태인 드라이브에서 남아 있는 데이터를 제거하는 방법입니다.  민감한 데이터가 담긴 드라이브의 수명이 다 되어가고 있다면, 드라이브를 교체할 때 이 방법을 사용할 수 있습니다.  이 방법은 보안 삭제 장치 명령을 사용하여 드라이브에 미리 정해진 패턴을 쓰고 드라이브의 암호화 키를 재설정합니다.  이 비동기 방식을 완료하는 데 몇 분이 걸릴 수 있습니다.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
드라이브
 a| 
안전하게 지울 드라이브 ID 목록입니다.
 a| 
정수 배열
 a| 
None
 a| 
예

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
비동기 핸들
 a| 
작업 결과를 얻는 데 사용되는 핸들 값입니다.
 a| 
정수

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "SecureEraseDrives",
   "params": {
     "drives"  : [3, 4, 5]
   },
   "id" : 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
   "id" : 1
   "result" : {
     "asyncHandle" : 1
   }
}
----
