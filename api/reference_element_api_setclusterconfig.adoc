= 클러스터 구성 설정
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `SetClusterConfig` 노드가 연관된 클러스터와 통신하는 데 사용하는 구성을 설정하는 방법입니다.  노드의 현재 클러스터 인터페이스 설정을 표시하려면 다음을 실행하세요. `GetClusterConfig` API 방식.



== 매개변수

이 메서드에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
무리
 a| 
이 메서드 호출 중에 변경되어야 하는 구성 속성입니다.  변경하려는 필드만 이 매개변수의 멤버로 이 메서드에 추가하면 됩니다.
 a| 
xref:reference_element_api_cluster.adoc[무리]
 a| 
None
 a| 
아니요

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
무리
 a| 
노드가 클러스터와 통신하는 데 사용하는 구성 정보입니다.
 a| 
xref:reference_element_api_cluster.adoc[무리]

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "SetClusterConfig",
   "params": {
      "cluster": {
         "name": "myhost",
         "mipi": "Bond10G"
      },
    "id" : 1
   }
}
----

}
----
