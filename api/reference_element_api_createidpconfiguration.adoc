= CreateIdpConfiguration
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `CreateIpdConfiguration` 클러스터에 대한 타사 ID 공급자(IdP)를 사용하여 인증을 위한 잠재적 신뢰 관계를 생성하는 방법입니다.  IdP 통신에는 SAML 서비스 공급자 인증서가 필요합니다.  이 인증서는 필요에 따라 생성되며, 이 API 호출을 통해 반환됩니다.



== 매개변수

이 방법에는 다음과 같은 입력 매개변수가 있습니다.

|===
| 이름 | 설명 | 유형 | 기본값 | 필수의 


 a| 
idp메타데이터
 a| 
저장할 IdP 메타데이터.
 a| 
끈
 a| 
None
 a| 
예



 a| 
IDP 이름
 a| 
SAML 2.0 Single Sign-On을 위한 IdP 공급자를 식별하는 데 사용되는 이름입니다.
 a| 
끈
 a| 
None
 a| 
예

|===


== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===


| 이름 | 설명 | 유형 


 a| 
idpConfigInfo
 a| 
타사 ID 공급자(IdP) 구성에 대한 정보입니다.
 a| 
link:reference_element_api_idpconfiginfo.html["idpConfigInfo"]

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
    "method": "CreateIdpConfiguration",
    "params": {
        "idpMetadata": "<?xml version=\"1.0\" encoding=\"UTF-8\"?>
             <EntityDescriptor
               xmlns=\"urn:oasis:names:tc:SAML:2.0:metadata\"
               xmlns:ds=\"http://www.w3.org/2000/09/xmldsig#\"
               xmlns:shibmd=\"urn:mace:shibboleth:metadata:1.0\"
               xmlns:xml=\"http://www.w3.org/XML/1998/namespace\"
               ...</Organization>
             </EntityDescriptor>",
        "idpName": "https://provider.name.url.com"
    },
}
----
