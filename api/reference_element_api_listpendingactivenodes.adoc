= 보류 중인 활성 노드 목록
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `ListPendingActiveNodes` 보류 중과 활성 상태 사이에 있는 PendingActive 상태에 있는 클러스터의 노드를 나열하는 방법입니다.  이 상태의 노드는 팩토리 이미지로 돌아갑니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
보류 중인 활성 노드
 a| 
시스템의 모든 PendingActive 노드에 대한 정보를 자세히 설명하는 개체 목록입니다.
 a| 
xref:reference_element_api_pendingactivenode.adoc[보류 중인 활성 노드]정렬

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
   "method": "ListPendingActiveNodes",
   "params": {},
   "id" : 1
}
----
