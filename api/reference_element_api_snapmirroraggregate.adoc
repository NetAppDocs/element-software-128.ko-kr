[role="lead"]
snapMirrorAggregate 개체에는 볼륨에 저장소로 사용할 수 있는 디스크 모음인 사용 가능한 ONTAP 집계에 대한 정보가 포함되어 있습니다.  ListSnapMirrorAggregates API 메서드를 사용하여 이 정보를 얻을 수 있습니다.
