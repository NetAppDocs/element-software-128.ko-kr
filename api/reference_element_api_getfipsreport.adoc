= GetFipsReport
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
당신은 사용할 수 있습니다 `GetFipsReport` 스토리지 클러스터의 모든 노드에 대한 FIPS 140-2 암호화 기능 지원 상태를 확인하는 방법입니다.



== 매개변수

이 방법에는 입력 매개변수가 없습니다.



== 반환 값

이 메서드는 다음과 같은 반환 값을 갖습니다.

|===
| 이름 | 설명 | 유형 


 a| 
결과
 a| 
모든 노드에 대한 FIPS 140-2 기능 지원 상태와 쿼리에 응답하지 않은 각 노드에 대한 오류 정보를 포함하는 JSON 개체입니다.
 a| 
xref:reference_element_api_fipsreport.adoc[fips보고서]

|===


== 요청 예시

이 방법에 대한 요청은 다음 예와 유사합니다.

[listing]
----
{
  "method": "GetFipsReport",
  "params": {},
  "id" : 1
}
----


== 응답 예시

이 메서드는 다음 예와 유사한 응답을 반환합니다.

[listing]
----
{
    "id": 1,
    "result": {
    "nodes": [
        {
           "nodeID": 1,
		   "fipsDrives": "None",
           "httpsEnabled": true
        },
        {
           "nodeID": 3,
		   "fipsDrives": "None",
           "httpsEnabled": true
        }
    ],
    "errorNodes": [
        {
           "nodeID": 2,
           "error": {
               "message": "The RPC timed out.",
               "name": "xRpcTimeout"
               }
        }
    ]
    }
}
----
