= 가상 네트워크 관리
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SolidFire 스토리지의 가상 네트워킹을 사용하면 별도의 논리적 네트워크에 있는 여러 클라이언트 간의 트래픽을 하나의 클러스터에 연결할 수 있습니다.  클러스터에 대한 연결은 VLAN 태깅을 통해 네트워킹 스택에서 분리됩니다.
