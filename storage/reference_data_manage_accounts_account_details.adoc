= 계정 세부 정보
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
관리 탭의 계정 페이지는 각 계정에 할당된 볼륨의 ID, 사용자 이름, 효율성 세부 정보 등 시스템의 각 계정에 대한 정보를 제공합니다.

* *ID*
+
계정에 대한 시스템 생성 ID입니다.

* *사용자 이름*
+
계정이 생성될 때 부여된 이름입니다.

* *상태*
+
계정의 상태. 가능한 값:

+
** 활성: 활성 계정.
** 잠김: 잠긴 계정.
** 제거됨: 삭제 및 정리된 계정입니다.


* *활성 볼륨*
+
계정에 할당된 활성 볼륨의 수입니다.

* *압축*
+
계정에 할당된 볼륨에 대한 압축 효율성 점수입니다.

* *중복 제거*
+
계정에 할당된 볼륨에 대한 중복 제거 효율성 점수입니다.

* *씬 프로비저닝*
+
계정에 할당된 볼륨에 대한 씬 프로비저닝 효율성 점수입니다.

* *전반적인 효율성*
+
계정에 할당된 볼륨에 대한 전반적인 효율성 점수입니다.
