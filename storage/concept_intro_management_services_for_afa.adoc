= SolidFire 올플래시 스토리지 관리 서비스
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
관리 서비스는 SolidFire 올플래시 스토리지에 대한 중앙 집중식 관리 기능과 확장된 관리 기능을 제공합니다.  이러한 서비스에는 Active IQ 시스템 원격 측정, 로깅, 서비스 업데이트는 물론 vCenter용 Element 플러그인을 위한 QoSSIOC 서비스가 포함됩니다.
