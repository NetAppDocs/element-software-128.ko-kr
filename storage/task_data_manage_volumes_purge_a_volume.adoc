= 볼륨 제거
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
볼륨이 제거되면 시스템에서 영구적으로 제거됩니다.  볼륨의 모든 데이터가 손실됩니다.

시스템은 삭제된 볼륨을 삭제한 후 8시간이 지나면 자동으로 삭제합니다.  하지만 예약된 시간 전에 볼륨을 정리하려는 경우에는 그렇게 할 수 있습니다.


NOTE: 한 번에 많은 수의 볼륨을 정리하거나 정리하는 각 볼륨에 연관된 스냅샷이 많은 경우 정리 프로세스가 실패하고 "xDBConnectionLoss" 오류가 반환될 수 있으며, 이 오류는 Element 웹 UI의 API 로그에 표시됩니다.  이런 일이 발생하면 볼륨을 줄여서 다시 제거를 시도하세요.

. *관리* > *볼륨*을 선택합니다.
. *삭제됨* 버튼을 클릭하세요.
. 단일 볼륨이나 여러 볼륨을 정리하는 단계를 수행합니다.
+
|===


| 옵션 | 단계 


 a| 
단일 볼륨을 제거합니다
 a| 
.. 삭제할 볼륨의 작업 아이콘을 클릭합니다.
.. *삭제*를 클릭하세요.
.. 작업을 확인합니다.




 a| 
여러 볼륨을 제거합니다
 a| 
.. 제거할 볼륨을 선택하세요.
.. *대량 작업*을 클릭하세요.
.. 나타나는 메뉴에서 *제거*를 선택합니다.
.. 작업을 확인합니다.


|===
