= 휴면 상태에서 암호화 활성화
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
클러스터 전체의 암호화를 저장 상태에서 활성화하거나 비활성화할 수 있습니다.  이 기능은 기본적으로 활성화되어 있지 않습니다.  FIPS 드라이브를 지원하려면 저장 데이터 암호화를 활성화해야 합니다.

. NetApp Element 소프트웨어 UI에서 *클러스터* > *설정*을 클릭합니다.
. *저장 시 암호화 사용*을 클릭합니다.
