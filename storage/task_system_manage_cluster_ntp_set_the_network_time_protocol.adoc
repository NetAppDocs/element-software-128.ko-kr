= 네트워크 시간 프로토콜 설정
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
네트워크 시간 프로토콜(NTP)을 설정하는 방법은 두 가지가 있습니다. 클러스터의 각 노드에 브로드캐스트를 수신하도록 지시하거나, 각 노드에 NTP 서버에 쿼리를 보내 업데이트를 요청하는 것입니다.

NTP는 네트워크 상에서 시계를 동기화하는 데 사용됩니다.  내부 또는 외부 NTP 서버에 대한 연결은 초기 클러스터 설정의 일부여야 합니다.
