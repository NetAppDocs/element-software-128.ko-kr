= 배포 후 SolidFire 시스템 옵션 구성
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SolidFire 시스템을 설정한 후에는 몇 가지 선택적 작업을 수행하고 싶을 수 있습니다.

시스템의 자격 증명을 변경하는 경우 다른 구성 요소에 미치는 영향을 알고 싶을 수 있습니다.

또한 다중 인증 요소, 외부 키 관리, FIPS(연방 정보 처리 표준) 보안에 대한 설정을 구성할 수 있습니다.  필요한 경우 비밀번호를 업데이트하는 것도 고려해 보세요.
