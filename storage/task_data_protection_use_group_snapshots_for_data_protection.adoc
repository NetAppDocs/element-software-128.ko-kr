= 데이터 보호 작업을 위한 그룹 스냅샷 사용
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
각 볼륨의 메타데이터에 대한 특정 시점 사본을 보존하기 위해 관련 볼륨 세트의 그룹 스냅샷을 만들 수 있습니다.  나중에 그룹 스냅샷을 백업이나 롤백으로 사용하여 볼륨 그룹의 상태를 이전 상태로 복원할 수 있습니다.
