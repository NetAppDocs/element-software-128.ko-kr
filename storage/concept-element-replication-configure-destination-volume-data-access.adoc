= 데이터 액세스를 위한 대상 볼륨 구성
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
대상 볼륨을 쓰기 가능하게 만든 후에는 데이터 액세스를 위해 볼륨을 구성해야 합니다.  SAN 호스트는 소스 볼륨이 다시 활성화될 때까지 대상 볼륨의 데이터에 액세스할 수 있습니다.

. Element LUN을 적절한 개시자 그룹에 매핑합니다.
. SAN 호스트 이니시에이터에서 SAN LIF로 iSCSI 세션을 생성합니다.
. SAN 클라이언트에서 스토리지 재스캔을 수행하여 연결된 LUN을 감지합니다.
