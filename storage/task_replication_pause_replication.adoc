= 복제 일시 중지
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
짧은 시간 동안 I/O 처리를 중지해야 하는 경우 복제를 수동으로 일시 중지할 수 있습니다.  I/O 처리가 급증하고 처리 부하를 줄이려는 경우 복제를 일시 중지하는 것이 좋습니다.

. *데이터 보호* > *볼륨 쌍*을 선택합니다.
. 볼륨 쌍에 대한 작업 아이콘을 클릭합니다.
. *편집*을 클릭하세요.
. *볼륨 쌍 편집* 창에서 복제 프로세스를 수동으로 일시 중지합니다.
+

IMPORTANT: 볼륨 복제를 수동으로 일시 중지하거나 다시 시작하면 데이터 전송이 중단되거나 다시 시작됩니다.  두 사이트 모두에서 이러한 변경 사항을 조정했는지 확인하세요.

. *변경 사항 저장*을 클릭하세요.
