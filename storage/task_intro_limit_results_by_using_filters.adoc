= 필터를 사용하여 결과 제한
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
Element UI에서 페이지의 목록 정보를 필터링할 수 있습니다.  목록(예: 볼륨 및 스냅샷)을 볼 때 하나 이상의 필터를 추가하여 정보에 초점을 맞추고 페이지에 더 쉽게 맞출 수 있습니다.

. 목록 정보를 볼 때 *필터*를 클릭하세요.
. *필터 기준* 필드를 확장합니다.
. 필드의 가장 왼쪽 요소에서 필터링할 열을 선택합니다.
. 열에 대한 제약조건을 선택합니다.
. 필터링할 텍스트를 입력하세요.
. *추가*를 클릭하세요.
+
시스템은 목록에 있는 정보에 대해 새로운 필터를 실행하고, 필터 기준 필드에 새로운 필터를 임시로 저장합니다.

. 다른 필터를 추가하려면 *추가*를 클릭하고 다시 필터를 선택하세요.
. *모두 지우기*를 클릭하면 필터 목록이 제거되고 필터링되지 않은 목록 정보가 표시됩니다.
