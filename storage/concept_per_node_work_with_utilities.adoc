= 스토리지 노드에 대한 노드별 유틸리티 사용
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
NetApp Element 소프트웨어 UI의 표준 모니터링 도구가 문제 해결에 필요한 충분한 정보를 제공하지 못하는 경우, 노드별 유틸리티를 사용하여 네트워크 문제를 해결할 수 있습니다.  노드별 유틸리티는 노드 간 또는 관리 노드와 관련된 네트워크 문제를 해결하는 데 도움이 되는 구체적인 정보와 도구를 제공합니다.
