= 피드백을 제공하세요
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
UI 전반에 걸쳐 접근 가능한 피드백 양식을 사용하면 Element 소프트웨어 웹 사용자 인터페이스를 개선하고 UI 문제를 해결하는 데 도움을 줄 수 있습니다.

. Element UI의 모든 페이지에서 *피드백* 버튼을 클릭하세요.
. 요약 및 설명 필드에 관련 정보를 입력합니다.
. 도움이 되는 스크린샷을 첨부해 주세요.
. 이름과 이메일 주소를 입력하세요.
. 현재 환경에 대한 데이터를 포함하려면 확인란을 선택하세요.
. *제출*을 클릭하세요.
