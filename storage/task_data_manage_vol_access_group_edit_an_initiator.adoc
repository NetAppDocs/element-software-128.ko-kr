= 개시자 편집
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
기존 개시자의 별칭을 변경하거나 별칭이 없으면 추가할 수 있습니다.

이니시에이터당 CHAP 계정 이름과 자격 증명을 추가하려면 다음을 사용해야 합니다. `ModifyInitiator` CHAP 액세스 및 속성을 제거하고 추가하는 API 호출입니다.

보다link:../api/concept_element_api_about_the_api.html["Element API로 저장소 관리"] .

.단계
. *관리* > *개시자*를 클릭합니다.
. 편집하려는 개시자의 작업 아이콘을 클릭합니다.
. *편집*을 클릭하세요.
. *별칭* 필드에 개시자의 새 별칭을 입력합니다.
. *변경 사항 저장*을 클릭하세요.
