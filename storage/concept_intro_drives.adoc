= 드라이브
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
저장 노드에는 클러스터의 데이터 일부를 저장하는 데 사용되는 하나 이상의 물리적 드라이브가 포함되어 있습니다.  클러스터는 드라이브가 클러스터에 성공적으로 추가된 후 드라이브의 용량과 성능을 활용합니다.

저장 노드에는 두 가지 유형의 드라이브가 포함됩니다.

* *볼륨 메타데이터 드라이브*
+
이러한 드라이브는 클러스터 내의 각 볼륨, 복제본 또는 스냅샷을 정의하는 압축 정보를 저장합니다.  시스템의 총 메타데이터 드라이브 용량은 볼륨으로 프로비저닝할 수 있는 최대 저장 용량을 결정합니다.  프로비저닝할 수 있는 최대 저장 용량은 클러스터의 블록 드라이브에 실제로 저장된 데이터 용량과 무관합니다.  볼륨 메타데이터 드라이브는 Double Helix 데이터 보호를 사용하여 클러스터 전체에 데이터를 중복하여 저장합니다.

+

NOTE: 일부 시스템 이벤트 로그와 오류 메시지에서는 볼륨 메타데이터 드라이브를 슬라이스 드라이브라고 합니다.

* *블록 드라이브*
+
이러한 드라이브는 서버 애플리케이션 볼륨을 위한 압축되고 중복이 제거된 데이터 블록을 저장합니다.  블록 드라이브는 시스템 저장 용량의 대부분을 차지합니다.  SolidFire 클러스터에 이미 저장된 데이터에 대한 대부분의 읽기 요청과 데이터 쓰기 요청은 블록 드라이브에서 발생합니다.  시스템의 총 블록 드라이브 용량은 압축, 씬 프로비저닝, 중복 제거의 효과를 고려하여 저장할 수 있는 최대 데이터 양을 결정합니다.
