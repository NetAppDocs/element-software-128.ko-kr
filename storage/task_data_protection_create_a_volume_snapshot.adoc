= 볼륨 스냅샷 생성
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
활성 볼륨의 스냅샷을 만들어 언제든지 볼륨 이미지를 보존할 수 있습니다.  단일 볼륨에 대해 최대 32개의 스냅샷을 만들 수 있습니다.

. *관리* > *볼륨*을 클릭합니다.
. 스냅샷에 사용할 볼륨의 *작업* 아이콘을 클릭합니다.
. 결과 메뉴에서 *스냅샷*을 선택합니다.
. *볼륨 스냅샷 만들기* 대화 상자에서 새 스냅샷 이름을 입력합니다.
. *선택 사항:* 부모 볼륨이 페어링될 때 복제에서 스냅샷이 캡처되도록 하려면 *페어링 시 복제에 스냅샷 포함* 확인란을 선택합니다.
. 스냅샷의 보존 기간을 설정하려면 다음 옵션 중 하나를 선택하세요.
+
** *영구 보관*을 클릭하면 스냅샷이 시스템에 무기한 보관됩니다.
** *보존 기간 설정*을 클릭하고 날짜 스핀 상자를 사용하여 시스템이 스냅샷을 보관할 기간을 선택합니다.


. 단일 스냅샷을 즉시 찍으려면 다음 단계를 수행하세요.
+
.. *지금 스냅샷 찍기*를 클릭하세요.
.. **스냅샷 만들기**를 클릭합니다.


. 스냅샷을 나중에 실행하도록 예약하려면 다음 단계를 수행하세요.
+
.. *스냅샷 일정 만들기*를 클릭하세요.
.. *새 일정 이름*을 입력하세요.
.. 목록에서 *일정 유형*을 선택하세요.
.. *선택 사항:* 예약된 스냅샷을 주기적으로 반복하려면 *반복 일정* 확인란을 선택합니다.
.. *일정 만들기*를 클릭하세요.
