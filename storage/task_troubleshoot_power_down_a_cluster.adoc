= 클러스터 전원 끄기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
전체 클러스터의 전원을 끄려면 다음 절차를 수행하세요.

.단계
. (선택 사항) 예비 단계를 완료하는 데 도움이 필요하면 NetApp 지원팀에 문의하세요.
. 모든 I/O가 중지되었는지 확인하세요.
. 모든 iSCSI 세션 연결 해제:
+
.. 클러스터의 관리 가상 IP(MVIP) 주소로 이동하여 Element UI를 엽니다.
.. 노드 목록에 나열된 노드를 확인하세요.
.. 클러스터의 각 노드 ID에 지정된 중지 옵션을 사용하여 Shutdown API 메서드를 실행합니다.




[NOTE]
====
클러스터를 다시 시작하면 모든 노드가 온라인 상태가 되었는지 확인하기 위해 다음 단계를 따라야 합니다.

. 모든 중요 심각도 및 `volumesOffline` 클러스터 오류가 해결되었습니다.
. 클러스터가 안정될 때까지 10~15분 정도 기다리세요.
. 데이터에 접근하기 위해 호스트를 시작합니다.


유지 관리 후 노드에 전원을 공급하고 노드가 정상인지 확인할 때 더 많은 시간을 허용하려면 불필요한 빈 동기화를 방지하기 위해 데이터 동기화를 지연시키는 방법에 대한 도움을 받기 위해 기술 지원팀에 문의하세요.

====
