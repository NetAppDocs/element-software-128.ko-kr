= 클러스터 전체 임계값 설정
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
아래 단계에 따라 시스템이 블록 클러스터 충만 경고를 생성하는 수준을 변경할 수 있습니다.  또한 ModifyClusterFullThreshold API 메서드를 사용하여 시스템이 차단 또는 메타데이터 경고를 생성하는 수준을 변경할 수 있습니다.

.필요한 것
클러스터 관리자 권한이 있어야 합니다.

.단계
. *클러스터* > *설정*을 클릭합니다.
. 클러스터 전체 설정 섹션에서 *Helix가 노드 장애에서 복구할 수 없게 되기 전에 _% 용량이 남을 때 경고 알림을 발생시킵니다*에 백분율을 입력합니다.
. *변경 사항 저장*을 클릭하세요.
