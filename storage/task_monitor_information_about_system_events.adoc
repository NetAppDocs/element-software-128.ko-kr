= 시스템 이벤트에 대한 정보 보기
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
시스템에서 감지된 다양한 이벤트에 대한 정보를 볼 수 있습니다.  시스템은 30초마다 이벤트 메시지를 새로 고칩니다.  이벤트 로그에는 클러스터의 주요 이벤트가 표시됩니다.

. Element UI에서 *보고* > *이벤트 로그*를 선택합니다.
+
모든 이벤트에 대해 다음 정보를 볼 수 있습니다.

+
|===


| 목 | 설명 


 a| 
ID
 a| 
각 이벤트와 관련된 고유 ID입니다.



 a| 
이벤트 유형
 a| 
기록되는 이벤트의 유형(예: API 이벤트 또는 복제 이벤트)입니다.



 a| 
메시지
 a| 
이벤트와 관련된 메시지입니다.



 a| 
세부
 a| 
이벤트가 발생한 이유를 식별하는 데 도움이 되는 정보입니다.



 a| 
서비스 ID
 a| 
이벤트를 보고한 서비스(해당되는 경우)



 a| 
마디
 a| 
이벤트를 보고한 노드(해당되는 경우)



 a| 
드라이브 ID
 a| 
이벤트를 보고한 드라이브(해당되는 경우).



 a| 
이벤트 시간
 a| 
사건이 발생한 시간.

|===
