= 가상 네트워크 삭제
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
가상 네트워크 개체를 제거할 수 있습니다.  가상 네트워크를 제거하기 전에 다른 가상 네트워크에 주소 블록을 추가해야 합니다.

. *클러스터* > *네트워크*를 클릭합니다.
. 삭제하려는 VLAN에 대한 작업 아이콘을 클릭합니다.
. *삭제*를 클릭하세요.
. 메시지를 확인하세요.
